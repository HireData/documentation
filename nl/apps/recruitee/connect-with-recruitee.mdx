_Welkom bij onze uitgebreide gids voor het koppelen van je Recruitee-account aan HireData. Deze tutorial is gebruiksvriendelijk en efficiënt, en helpt je het volledige potentieel van Recruitee binnen je HireData-automatiseringen te benutten. Bekijk voor een visuele uitleg vooral onze stapsgewijze **Connect with Recruitee Video** aan het einde._

## Stap 1: Navigeren naar de Recruitee-app

Begin met navigeren naar de [Apps-sectie](https://app.hiredata.com/settings/apps) binnen HireData. Je vindt dit onder Settings &gt; Apps. Hier zie je de optie om te [integreren met Recruitee.](https://app.hiredata.com/apps/recruitee) Deze eerste stap is eenvoudig maar essentieel voor het integratieproces.

![](/images/intercom/65643328-Schermafbeelding-2B2025-09-19-2B100347.png)

## Stap 2: Een verbinding met Recruitee maken

Open de app en klik op **Connect to Recruitee** om het integratieproces te starten. Hiermee kan HireData verbinding maken met je Recruitee-account.

![](/images/intercom/65646660-Schermafbeelding-2B2025-09-19-2B102104.png)

### 2.1. API token aanmaken

Klik op '**here'** om een nieuwe API token aan te maken in Recruitee.

Zorg dat je gebruiker deze rollen heeft:

- Basic permissions → Collaborate on candidates, Join hiring teams
- Jobs → View all existing jobs data
- Hires → View hired candidates
- Add-ons → Manage API tokens

Kopieer nu de gegenereerde API token en plak deze hieronder om Recruitee te koppelen aan HireData.

![](/images/intercom/65646659-image.png)

### 2.2. Kopieer je Recruitee Company ID

Onder **Current company details in Recruitee** zie je je **Company ID**. Kopieer dit ID gewoon en plak het in het veld in HireData.

![](/images/intercom/65646661-image.png)

Zodra dit is ingevuld, klik je op [Continue] om de verbinding met Recruitee te voltooien.

![](/images/intercom/66842172-image.png)

## Stap 3: Recruitee verbonden!

Nadat je op [Continue] hebt geklikt, is het opzetten van de verbinding tussen Recruitee en HireData voltooid. Wanneer je deze verbinding voor het eerst hebt opgezet, kan HireData enkele minuten nodig hebben om data uit je Recruitee-database te synchroniseren. Daarna ben je klaar om [je eerste Recruitee-automatisering op te zetten](/nl/apps/recruitee/create-a-recruitee-automation-in-hiredata) in HireData.

![](/images/intercom/66841878-image.png)
