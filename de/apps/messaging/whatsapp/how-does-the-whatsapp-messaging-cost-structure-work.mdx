**_Zu verstehen, wie die Kosten für WhatsApp-Messaging strukturiert sind, ist entscheidend für die effektive Verwaltung Ihres Kommunikationsbudgets. Nachfolgend finden Sie eine klare Aufschlüsselung, wie die Kosten in Ihrem HireData-Setup funktionieren._**

# Eingehende Nachrichten sind kostenlos

Wenn ein Kandidat eine Konversation initiiert, indem er eine Nachricht an Ihre WhatsApp-Nummer sendet (z. B. Bewerbung über WhatsApp), haben Sie ein 24-Stunden-Fenster, um zu antworten, ohne dass zusätzliche Gebühren von Meta anfallen.

# Ausgehende Nachrichten erfordern Vorlagen

Wenn Sie proaktiv eine neue Konversation starten (eine ausgehende Nachricht senden), müssen Sie eine genehmigte WhatsApp-Vorlage verwenden. Diese Vorlagen sind in der Regel kostenpflichtig, insbesondere wenn sie in die Marketing-Kategorie fallen.

# Vorlagenkosten werden von Meta abgerechnet

WhatsApp-Vorlagenkosten werden direkt von Meta (Facebook) abgerechnet, basierend auf dem Nachrichtentyp und der Region, in die gesendet wird. Diese Gebühren erscheinen separat auf Ihrer Meta-Rechnung.

Metas detaillierte Preise finden Sie hier: [WhatsApp Business Platform Pricing](https://business.whatsapp.com/products/platform-pricing).

# Credit-Abzug innerhalb von HireData

Zusätzlich zu Metas Gebühren verbraucht jede automatisierte Aufgabe Credits aus Ihrem HireData-Plan. Dies ist Teil der Monetarisierungsstruktur von HireData. Das heißt, wenn Sie eine WhatsApp-Nachricht senden oder einen Agenten eine Konversation führen lassen, werden sowohl von Ihrem Meta-Budget als auch von Ihren HireData-Credits Beträge abgezogen.

# Automatisierungsfreundliche Struktur

Wenn der Kandidat Sie bereits kontaktiert hat, berechnet Meta innerhalb der nachfolgenden 24 Stunden keine Gebühren für automatisierte Antworten. Nach diesem 24-Stunden-Fenster müssen Sie eine kostenpflichtige WhatsApp-Vorlage verwenden, um die Konversation fortzusetzen.

# Sichtbarkeit der Kosten

Sie können Ihre WhatsApp-bezogenen Ausgaben über Metas Transaktions-Interface überwachen und nachverfolgen, um vollständige Transparenz zu erhalten.

Zusammenfassend:

- Antworten innerhalb von 24 Stunden auf eingehende Nachrichten sind kostenlos.
- Proaktive ausgehende Nachrichten erfordern kostenpflichtige Vorlagen.
- Die Kosten umfassen sowohl Meta-Gebühren als auch Abzüge von HireData-Credits.
