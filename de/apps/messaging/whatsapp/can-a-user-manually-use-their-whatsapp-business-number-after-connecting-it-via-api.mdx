_Dieser Artikel erklärt, wie Benutzer ihre WhatsApp-Konversationen effektiv über die Inbox von HireData verwalten können, sobald eine WhatsApp-Business-Account-Nummer (WABA) über die WhatsApp Business API verbunden ist._

## Was passiert, wenn Sie ein WABA über die API verbinden

Sobald eine WhatsApp-Business-Nummer mit der WhatsApp Business API integriert ist:

- Kann die Nummer **nicht** über die Standard-WhatsApp- oder WhatsApp-Business-Mobilanwendungen genutzt werden.
- Erfolgt die gesamte Nachrichtenverarbeitung – ein- und ausgehend – ausschließlich über die Inbox-Oberfläche von HireData.

## Ihre Konversationen in der HireData-Inbox verwalten

### Bei HireData anmelden

- Navigieren Sie zu Ihrem HireData-Dashboard und wählen Sie im Menü die **Inbox**.

![HireData inbox](/images/intercom/51618039-Screenshot-2025-05-28-at-15_47_46.png)

### Eingehende Nachrichten bearbeiten

- Neue WhatsApp-Nachrichten erscheinen in Echtzeit in der HireData-Inbox.
- Konversationen sind klar gekennzeichnet und mit dem relevanten Kontakt, Benutzer, der Marke oder dem Konto verknüpft.

![Message in HireData inbox](/images/intercom/51618089-Screenshot-2025-05-28-at-15_48_17.png)

### Nachrichten senden

- Sie können manuell direkt aus der Inbox auf eingehende Nachrichten antworten.
- Aus der HireData-Inbox gesendete Nachrichten erscheinen für Empfänger genauso, als wären sie direkt von Ihrer verbundenen WhatsApp-Business-Nummer gesendet worden.

### Automatisierung und Personalisierung

- Erste Outreach-Nachrichten und automatisierte Erinnerungen können so eingerichtet werden, dass sie automatisch ausgeführt werden.
- Benutzer können automatisierte Konversationen jederzeit nahtlos übernehmen, um personalisierte Kommunikation sicherzustellen.

## Vorteile der HireData-Inbox

- **Personalisiertes Outreach:** Über HireData gesendete Nachrichten spiegeln Ihre verbundene WhatsApp-Business-Nummer wider und wahren eine personalisierte Ansprache.
- **Zentralisierte Kommunikation:** Verwalten Sie alle Konversationen einfach über eine einzige Oberfläche.
- **Automatisierungskontrolle:** Kombinieren Sie automatisiertes Outreach nach Belieben mit manuellen Follow-ups.

### Mobile Verfügbarkeit

Derzeit ist die Inbox von HireData über Desktop-Browser zugänglich. Mobile Zugänglichkeit für die manuelle Nachrichtenbearbeitung wird für zukünftige Entwicklungen in Betracht gezogen.

## Best Practices

- Prüfen Sie Ihre HireData-Inbox regelmäßig für zeitnahe Follow-ups.
- Passen Sie Ihre automatisierten Sequenzen an, während Sie sich die Möglichkeit bewahren, jederzeit manuell einzugreifen.
- Kommunizieren Sie klar mit Ihren Empfängern, dass die erhaltenen Nachrichten von Ihrer verbundenen WhatsApp-Business-Nummer stammen.
