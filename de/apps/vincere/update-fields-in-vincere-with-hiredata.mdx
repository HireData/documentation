_Willkommen zu unserer Schritt-für-Schritt-Anleitung, wie Sie ein oder mehrere Felder in verschiedenen Objekten in Vincere mit HireData aktualisieren. Folgen Sie dieser einfachen, aber hilfreichen Anleitung, um zu verstehen, wie Sie jedes Feld selbst aktualisieren können._

Bevor Sie die Felder im Objekt aktualisieren, sollten Sie bereits eine [Automatisierung in Vincere](/de/apps/vincere/create-a-vincere-automation-in-hiredata) erstellt haben.

## Schritt 1: Das Building-Blocks-Menü öffnen

Melden Sie sich zunächst in Ihrem HireData-Konto an und öffnen Sie die Automatisierung, die Sie bearbeiten möchten. Klicken Sie dann am Ende der Automatisierung auf das Plus-Symbol [+], um das Building-Blocks-Menü auf der rechten Bildschirmseite zu öffnen. Nachdem Sie das Menü geöffnet haben, wählen Sie [Update].

![Building Blocks menu](/images/intercom/51931242-Screenshot-2025-06-02-at-09_16_11.png)

## Schritt 2: Das Objekt auswählen, das Sie bearbeiten möchten

Wählen Sie das Objekt aus, das Sie bearbeiten möchten. Es kann derzeit eines davon sein:

- Candidate
- Company
- Contact
- Job
- Meeting
- Task

_In diesem Beispiel haben wir den Candidate ausgewählt:_

![Selecting an object to be updated](/images/intercom/51931309-Screenshot-2025-06-02-at-09_19_18.png)

## Schritt 3: Einen Identifier auswählen

Nun müssen Sie einen Identifier auswählen, der verwendet wird, um das Objekt zu finden, das Sie aktualisieren möchten.

![Selecting an object identifier](/images/intercom/51931462-Screenshot-2025-06-02-at-09_19_42.png)

Für diese Anleitung verwenden wir die [ID] als Identifier und den [Candidate: ID] als den Wert, den wir verfolgen möchten. Mit anderen Worten: Wir verwenden die Candidate-ID, um den Kandidaten zu verfolgen, den wir in der Automatisierung aktualisieren möchten.

**_Die angezeigten Werte unterscheiden sich je nach ausgewähltem Objekt und Identifier._**

![Selecting object identifier](/images/intercom/51931552-Screenshot-2025-06-02-at-09_19_50.png)

## Schritt 4: Die Felder aktualisieren

Nun können Sie alle Felder auswählen, die Sie aktualisieren möchten – Sie können beliebig viele auswählen; nutzen Sie einfach die Suchleiste, um sie zu finden.

![Selecting field to be updated](/images/intercom/51931636-Screenshot-2025-06-02-at-09_20_14.png)

Bearbeiten Sie die Felder und klicken Sie unten auf der Seite auf die Schaltfläche [Save Task].

![Saving the update fields task](/images/intercom/51931666-Screenshot-2025-06-02-at-09_22_04.png)

## Schritt 5: Überprüfen

Zu guter Letzt prüfen Sie das Task-Popup, das am Ende der Automatisierung erscheint, um sicherzustellen, dass alles reibungslos läuft.

![Update field task](/images/intercom/51931722-Screenshot-2025-06-02-at-09_22_36.png)
