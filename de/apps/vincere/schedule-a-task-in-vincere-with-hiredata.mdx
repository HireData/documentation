_Die Aktion Vincere Scheduled Task ermöglicht es Ihnen, in Vincere automatisch geplante Aufgaben zu erstellen, wann immer ein bestimmtes Trigger-Ereignis in einem HireData-Automatisierungs-Workflow eintritt. Das ist nützlich, um Erinnerungen und Follow-up-Aktivitäten zu automatisieren, damit keine Aufgabe übersehen wird. Folgen Sie uns, während wir zeigen, wie Sie diese Aufgabe verwenden._

## Schritt 1: Eine Automatisierung erstellen oder bearbeiten

Navigieren Sie in Ihrem HireData-Konto zum Bereich **Automations** und erstellen Sie entweder einen neuen Workflow oder öffnen Sie einen bestehenden, indem Sie auf **+ New Automation** klicken.

![Adding new automation in HireData](/images/intercom/70618331-image.png)

## Schritt 2: Die Aktion hinzufügen

### 2.1. Klicken Sie auf **Blank Automation**, um eine neue Automatisierung von Grund auf zu starten

Die Auswahl von Blank Automation erstellt eine neue Automatisierung

![Start a blank automation](/images/intercom/73792029-image.png)

### 2.2. Klicken Sie in den ersten Building Block/Trigger, um das App-Panel zu öffnen, und wählen Sie **Vincere**

Wenn Sie auf den ersten Building Block klicken, können Sie auswählen, mit welcher App dies verknüpft wird

![First step to creating a Trigger by selecting the brand](/images/intercom/72641866-image.png)

### 2.3. Wählen Sie den Trigger für die Automatisierung

Folgen Sie den Einrichtungsschritten, um festzulegen, wie diese Automatisierung ausgelöst werden soll, und klicken Sie auf **Save**.

Auswählen des Object-Wertes für die App

![Trigger Setup Choosing Object](/images/intercom/72645601-image.png)

Wählen Sie das Timing, dem der Trigger folgen soll

![Trigger Setup Timing](/images/intercom/72645827-image.png)

Prüfen, dass alles korrekt ist, und den Trigger speichern

![Overview of Trigger](/images/intercom/72646002-image.png)

### 2.4. Klicken Sie auf den 2. Building Block, um das Aktions-Panel zu öffnen, und klicken Sie auf **Add**, um eine neue Aufgabe zu erstellen

![](/images/intercom/72648002-image.png)

### 2.5. Navigieren Sie, bis Sie das Vincere-Logo zusammen mit dem Text **Task** sehen

![Add Task](/images/intercom/72648586-image.png)

## Schritt 3: Die Aufgabendetails ausfüllen

- **Type:** Wählen Sie den Item Type aus, dem die Aufgabe entsprechen soll. Das kann sein:
  - Candidate – Eine Aufgabe im Zusammenhang mit dem Job Seeker
  - Contact – Eine Aufgabe, die an den Client Contact gebunden ist
  - Job – Eine Aufgabe im Zusammenhang mit einem Job Order oder einer Vakanz
  - Personal – Eine Aufgabe für Ihre eigene Verwendung, z. B. Erinnerungen oder Admin-Aufgaben
  - Project – Eine Aufgabe, die mit einem größeren Projekt oder einer Kampagne verknüpft ist

![](/images/intercom/72656330-image.png)

- **Subject:** Geben Sie einen klaren und beschreibenden Betreff für die Aufgabe ein (z. B. „Test project, local")
- **Description:** Geben Sie eine detaillierte Beschreibung der Aufgabe an
- **Start/End Date:** Legen Sie das Start- und Enddatum der Aufgabe fest. Sie können auch eine Verzögerung setzen, wenn Sie möchten
- **Reminders:** Fügen Sie eine Erinnerung hinzu, damit die Aufgabe nicht vergessen wird
- **Assignee:** Weisen Sie die Aufgabe dem relevanten Benutzer in Vincere zu
![Task configuration](/images/intercom/70620488-image.png)

## Schritt 4: Speichern und testen

Klicken Sie auf **Save Task**, um die Aufgabenkonfiguration abzuschließen, und [führen Sie den Automatisierungs-Workflow zum Testen aus](/de/automations/how-to-test-an-automation).

## Schritt 5: Die Aufgabe bestätigen

- **HireData** – Navigieren Sie zu **Task History**, um zu prüfen, ob sie erstellt wurde.

![Tracking the task in HireData](/images/intercom/70620685-image.png)

- **Vincere** – Öffnen Sie **Email View → Task Tab**.

![Tracking the task in Vincere](/images/intercom/70620741-image.png)
