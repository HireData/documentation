_Willkommen zu unserer umfassenden Anleitung, wie Sie eine „Schedule task" in Carerix mit HireData erstellen. Folgen Sie den nachstehenden Anweisungen, um den Prozess zu erlernen und ihn selbst zu verwalten. Am Ende finden Sie außerdem ein Video-Tutorial zur zusätzlichen Unterstützung._

Sie sollten bereits eine Automatisierung für Carerix erstellt haben – wie das geht, sehen Sie [hier](/de/apps/carerix/create-a-carerix-automation-in-hiredata).

## Schritt 1: Eine Aufgabe erstellen

Klicken Sie auf [+], um das Building-Blocks-Menü zu öffnen, dann auf [Task] und wählen Sie schließlich die App [Carerix]. Sie sehen alle automatisierten Aufgaben, die wir derzeit über Carerix unterstützen:

- Add Activity or Item
- Update Field(s)
- Schedule task

Wählen Sie die Aufgabe aus, die Sie erstellen möchten.

![Building Blocks menu with Carerix automated tasks](/images/intercom/244c3d20ce45.png)

## Schritt 2: Einrichten

In dieser Anleitung konzentrieren wir uns auf das Erstellen einer „Schedule task". Nachdem Sie auf die Schaltfläche [Schedule Task] geklickt haben, müssen Sie einige Felder ausfüllen.

![Task settings](/images/intercom/75885f020ae2.png)

**_2.1. Den Owner der Aufgabe wählen_**

Als Erstes nach dem Erstellen der Aufgabe müssen Sie sicherstellen, dass der Owner mit dem Owner des Matches übereinstimmt. Wählen Sie dazu im Feld [Match Owner] die Option [ID] und anschließend im Feld [With] die Option [Match: Owner (User)]. So wird die Aufgabe dieser Person zugewiesen.

![Matching the owner setting](/images/intercom/720df4ec3271.png)

**_2.2. Start- und Fälligkeitsdatum festlegen_**

Je nachdem, wann die Aufgabe beginnen und abgeschlossen sein soll, können Sie ein [Start date] sowie ein [Due Date] festlegen.

![Start and due date options](/images/intercom/898d7cee1844.png)

**_2.3. Einen Betreff einfügen_**

Im Bereich [Subject] können Sie Variablen einfügen. Dafür gibt es zwei Möglichkeiten. Die erste besteht darin, die Variablen auf der rechten Seite über die Schaltfläche **\{\}** zu öffnen. Tippen Sie dann ein Stichwort der Variable ein, um sie schneller zu finden, und klicken Sie einfach darauf, um sie zu kopieren.

![Variables list](/images/intercom/16948b32fa7d.png)

Die andere Möglichkeit ist, im Betrefffeld „\{\{" gefolgt vom Namen der Variable einzugeben.

![Finding a variable in the subject field](/images/intercom/9520268e2ce8.png)

Wenn Sie zum Beispiel prüfen möchten, ob der Kandidat zufrieden ist, tippen Sie „Check if \{\{CandidateFirstName is happy".

![Subject example](/images/intercom/35af7e599608.png)

**_2.4. Eine Beschreibung verfassen_**

Schreiben Sie eine kurze Beschreibung der Aufgabe – sie sollte nicht länger als ein Satz sein.

![Description example](/images/intercom/2849eab1e7a5.png)

**_2.5. Den Status der Aufgabe auswählen_**

Er kann sein:

- To do
- In progress
- Done
- Screening accepted
- Screening declined

![Task status options](/images/intercom/d0e5894aa343.png)

**_2.6. Ablaufdatum oder Verzögerung hinzufügen_**

Dies ist optional. Wenn Sie sie nicht hinzufügen möchten, können Sie diesen Teil überspringen.

![Expiry date and delay settings](/images/intercom/b17544b14cd9.png)

**_2.6.1. Ein Ablaufdatum für die Aufgabe festlegen_**

![Expiry date options](/images/intercom/17be4d5d7a8a.png)

**_2.6.2. Eine Verzögerung für die Aufgabe festlegen_**

![Delay options](/images/intercom/93b6beee5bea.png)

**_2.7. Den Kandidaten zuordnen_**

Achten Sie darauf, den richtigen Kandidaten dieser Aufgabe zuzuordnen, damit sie auch zur Datei des Kandidaten in Carerix passt.

![Match Candidate field](/images/intercom/d9a4c4b18811.png)

**_2.8. Die Aufgabe speichern_**

Klicken Sie unten im Building-Blocks-Menü auf [Save Task].

![Save Task button](/images/intercom/a468ea5bb9fc.png)
