_Willkommen zu unserer Schritt-für-Schritt-Anleitung, wie Sie ein oder mehrere Felder in verschiedenen Objekten in Carerix mit HireData aktualisieren. Folgen Sie dieser einfachen, aber hilfreichen Anleitung, um zu verstehen, wie Sie jedes Feld selbst aktualisieren können. Am Ende finden Sie außerdem ein Video-Tutorial zur weiteren Orientierung._

Bevor Sie die Felder im Objekt aktualisieren, sollten Sie bereits eine [Automatisierung in Carerix](/de/apps/carerix/create-a-carerix-automation-in-hiredata) erstellt haben.

## Schritt 1: Das Building-Blocks-Menü öffnen

Melden Sie sich zunächst in Ihrem HireData-Konto an und öffnen Sie die Automatisierung, die Sie bearbeiten möchten. Klicken Sie dann am Ende der Automatisierung auf das Plus-Symbol [+], um das Building-Blocks-Menü zu öffnen. Es wird auf der rechten Bildschirmseite geöffnet.

![Building Blocks menu](/images/intercom/01019da70775.png)

## Schritt 2: Das Objekt auswählen, das Sie bearbeiten möchten

Nachdem Sie das Menü geöffnet haben, wählen Sie [Task], dann [Carerix] und wählen Sie eine Verbindung aus. Sie sehen dann alle Aufgaben, die wir unterstützen. Eine davon ist [Update Field(s)]. Klicken Sie darauf.

![Carerix tasks in HireData](/images/intercom/c7f808110c10.png)

An diesem Punkt müssen Sie das Objekt auswählen, das Sie bearbeiten möchten. Es kann eines davon sein:

- Appointment
- Candidate
- Company
- Match
- Placement
- Publication
- User
- Vacancy

_In diesem Beispiel haben wir den Candidate ausgewählt:_

![](/images/intercom/e72971e676f5.png)

## Schritt 3: Einen Identifier auswählen

Nun müssen Sie einen Identifier auswählen, der verwendet wird, um das Objekt in der Automatisierung zu finden. Dies könnte entweder eine ID oder eine Business/Private-E-Mail-Adresse sein.

![Identifier options](/images/intercom/78a1fea19371.png)

Für diese Anleitung verwenden wir die [ID] als Identifier und den [Candidate: ID] als den Wert, den wir verfolgen möchten. Mit anderen Worten: Wir verwenden die Candidate-ID, um den Kandidaten zu verfolgen, den wir in der Automatisierung aktualisieren möchten.

**_Die angezeigten Werte unterscheiden sich je nach ausgewähltem Objekt und Identifier._**

![Value options](/images/intercom/46632c22c475.png)

## Schritt 4: Die Felder aktualisieren

Nun können Sie alle Felder auswählen, die Sie aktualisieren möchten – Sie können beliebig viele auswählen; nutzen Sie einfach die Suchleiste, um sie zu finden.

![Choosing which fields to update](/images/intercom/131738d4ae83.png)

Bearbeiten Sie die Felder und klicken Sie unten auf der Seite auf die Schaltfläche [Save Task].

_Dies ist ein Beispiel für einige der Felder, die Sie aktualisieren können:_

![Example fields that can be updated](/images/intercom/6ecf00c5c07d.png)

## Schritt 5: Überprüfen

Zu guter Letzt prüfen Sie das Task-Popup, das am Ende der Automatisierung erscheint, um sicherzustellen, dass alles reibungslos läuft.

![Task summary](/images/intercom/38e498a63076.png)
