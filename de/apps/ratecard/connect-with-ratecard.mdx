_Willkommen zu unserer detaillierten Anleitung zur Verbindung Ihres Ratecard-Kontos mit HireData. Diese Anleitung ist benutzerfreundlich und effizient gestaltet und hilft Ihnen, das volle Potenzial von Ratecard innerhalb Ihrer HireData-Workflows auszuschöpfen. Für eine visuelle Anleitung sehen Sie sich unbedingt unser Schritt-für-Schritt-Video an._

## **Schritt 1: Zur Ratecard-App navigieren**

Beginnen Sie mit dem Aufruf der [App-Sektion](https://app.hiredata.com/apps/) in HireData. Dort finden Sie die [Ratecard-App](https://app.hiredata.com/apps/ratecard). Die Auswahl von Ratecard ist Ihr erster Schritt zur Integration.

## **Schritt 2: Eine neue Verbindung erstellen**

Sobald Sie Ratecard ausgewählt haben, klicken Sie auf die Schaltfläche [+ New Connection]. Diese Aktion startet den Prozess zur Verknüpfung Ihres Ratecard-Kontos mit HireData.

_Klicken Sie auf [+ New Connection] und beginnen Sie mit der Verbindung Ihres Ratecard_

![](/images/intercom/92061536df85.png)

## **Schritt 3: Access Token einholen**

Um fortzufahren, holen Sie sich ein Access Token, indem Sie auf den in HireData angegebenen Link klicken. Dieses Token ist entscheidend, um eine sichere Verbindung zwischen Ihrem Ratecard- und HireData-Konto herzustellen.

_Geben Sie Ihr Access Token an, um eine neue Verbindung einzurichten_

![](/images/intercom/06c1e112f9c9.png)
