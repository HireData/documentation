Modifiers ändern, wie ein Wert einer [Variable](/de/variables/introduction) beim Senden Ihres Inhalts erscheint. Fügen Sie einen Modifier mit einem Pipe (`|`) nach dem Feld hinzu.

## Wie Modifiers funktionieren

Ein Modifier steht nach dem Feld, getrennt durch ein Pipe:

```
{{ candidate.first_name | uppercase }} → SOFIA
```

Fügen Sie weitere Pipes hinzu, um Modifiers der Reihe nach von links nach rechts anzuwenden:

```
{{ candidate.first_name | lowercase | capitalise }} → sofia → Sofia
```

Einige Modifiers nehmen zusätzliche Werte nach einem Doppelpunkt an, getrennt durch Kommas:

```
{{ job.title | truncate_chars: 20 }} → Senior Software Engin...
{{ description | replace: "old", "new" }} → This is my new job
```

Argumente können sein:

- **Text** — in Anführungszeichen setzen: `"EUR"`, `"there"`
- **Zahlen** — einfach schreiben: `20`, `2`
- **Ein anderes Feld** — per Key referenzieren: `highlight: candidate.first_name`

<Tip>
  Viele Modifiers haben kürzere Aliase. Zum Beispiel funktioniert `upper` genauso wie
  `uppercase`, und `gt` wie `greater_than`. Aliase werden in den Tabellen unten in
  Klammern angezeigt.
</Tip>

## Fallback

| Modifier         | Was er macht                                     | Beispiel                                   |
| ---------------- | ------------------------------------------------ | ------------------------------------------ |
| `default: value` | Zeigt `value`, wenn das Feld leer oder nicht vorhanden ist | `{{ name \| default: "there" }}` → `there` |

## Text

| Modifier                              | Was er macht                                   | Beispiel                                              |
| ------------------------------------- | ---------------------------------------------- | ----------------------------------------------------- |
| `uppercase` (`upper`)                 | Wandelt den Text in Großbuchstaben um          | `sofia` → `SOFIA`                                     |
| `lowercase` (`lower`)                 | Wandelt den Text in Kleinbuchstaben um         | `SOFIA` → `sofia`                                     |
| `capitalise`                          | Schreibt jedes Wort groß                       | `john doe` → `John Doe`                               |
| `title_case`                          | Schreibt nur das erste Wort groß               | `john doe` → `John doe`                               |
| `camel_case`                          | Formatiert als camelCase                       | `john doe` → `johnDoe`                                |
| `pascal_case`                         | Formatiert als PascalCase                      | `john doe` → `JohnDoe`                                |
| `snake_case`                          | Verbindet die Wörter mit Unterstrichen         | `john doe` → `john_doe`                               |
| `kebab_case`                          | Verbindet die Wörter mit Bindestrichen         | `john doe` → `john-doe`                               |
| `dot_case`                            | Verbindet die Wörter mit Punkten               | `john doe` → `john.doe`                               |
| `slug`                                | Erstellt eine webfreundliche Version für Links | `John Doe!` → `john-doe`                              |
| `initials`                            | Nimmt die Initialen                            | `John Doe` → `JD`                                     |
| `trim`                                | Entfernt Leerzeichen an beiden Enden           | `  hi  ` → `hi`                                       |
| `trim_start`                          | Entfernt führende Leerzeichen                  | `  hi` → `hi`                                         |
| `trim_end`                            | Entfernt nachfolgende Leerzeichen              | `hi  ` → `hi`                                         |
| `append: text`                        | Fügt Text am Ende an                           | `Doe` + `append: "!"` → `Doe!`                        |
| `prepend: text`                       | Fügt Text am Anfang an                         | `Doe` + `prepend: "Mr "` → `Mr Doe`                   |
| `concat: a, b, …` (`combine`)         | Verkettet den Wert mit einem oder mehreren Texten oder Feldern | `first \| concat: " ", last` → `John Doe`           |
| `replace: find, with`                 | Ersetzt alle Treffer                           | `replace: "o", "0"` auf `john` → `j0hn`               |
| `replace_first: find, with`           | Ersetzt den ersten Treffer                     | `j0hn doe`                                            |
| `replace_last: find, with`            | Ersetzt den letzten Treffer                    | `john d0e`                                            |
| `remove: text`                        | Entfernt alle Treffer                          | `remove: " "` auf `john doe` → `johndoe`              |
| `remove_first: text`                  | Entfernt den ersten Treffer                    |                                                       |
| `remove_last: text`                   | Entfernt den letzten Treffer                   |                                                       |
| `truncate_chars: length`              | Kürzt auf eine Anzahl an Zeichen und ergänzt `…` | `truncate_chars: 5` auf `Engineering` → `Engin...`  |
| `truncate_words: count`               | Kürzt auf eine Anzahl an Wörtern und ergänzt `…` | `truncate_words: 2` auf `one two three` → `one two...` |
| `excerpt: phrase`                     | Zeigt einen Ausschnitt rund um einen passenden Ausdruck |                                              |
| `substring: start, length` (`substr`) | Nimmt einen Teil des Textes nach Position      | `substring: 0, 5` auf `Engineering` → `Engin`         |
| `word: index`                         | Nimmt ein einzelnes Wort (beginnend bei 0)     | `word: 1` auf `one two three` → `two`                 |
| `word_count` (`words`)                | Zählt die Wörter                               | `one two three` → `3`                                 |
| `length` (`len`, `size`, `count`)     | Zählt die Zeichen                              | `Sofia` → `5`                                         |
| `lines`                               | Teilt Text in separate Zeilen                  |                                                       |
| `wrap: width`                         | Umbricht Text auf eine Zeilenbreite            |                                                       |
| `pad_left: length, char`              | Füllt den Anfang auf eine Länge auf            | `pad_left: 3, "0"` auf `5` → `005`                    |
| `pad_right: length, char`             | Füllt das Ende auf eine Länge auf              | `pad_right: 3, "x"` auf `5` → `5xx`                   |
| `repeat: times`                       | Wiederholt den Text                            | `repeat: 3` auf `ab` → `ababab`                       |
| `mask: char, start, length`           | Verbirgt Teile des Textes                      | `mask` auf `secret` → `******`                        |
| `redact`                              | Verbirgt sensible Daten und behält einen Hinweis | `john@acme.com` → `j***@acme.com`                   |
| `highlight: term`                     | Hebt passende Wörter im Text hervor            | hebt `term` im Text hervor                            |
| `ascii` (`transliterate`)             | Ersetzt Akzentbuchstaben durch normale         | `José` → `Jose`                                       |
| `escape_html`                         | Zeigt HTML-Zeichen als Klartext                | `<b>` → `&lt;b&gt;`                                   |
| `strip_html`                          | Entfernt alle HTML-Tags                        | `<b>hi</b>` → `hi`                                    |
| `markdown_to_html`                    | Wandelt Markdown in formatierten Text um       | `**hi**` → `<strong>hi</strong>`                      |
| `sprintf: format` (`format`)          | Formatiert Text oder Zahlen mit einem Muster   | `sprintf: "%05.2f"` auf `3.1` → `03.10`               |

## Kontakt und Locale

| Modifier               | Was er macht                                                                          | Beispiel                                            |
| ---------------------- | ------------------------------------------------------------------------------------- | -------------------------------------------------- |
| `link`                 | Wandelt einen Wert in einen anklickbaren Link (E-Mail → `mailto:`, Telefon → `tel:`, Web → Anchor) | `a@b.com` → `<a href="mailto:a@b.com">a@b.com</a>` |
| `format_phone: region` | Formatiert eine Telefonnummer international                                            | `06 12345678` → `+31612345678`                     |
| `country_name`         | Zeigt den Ländernamen zu einem Code                                                    | `NL` → `Netherlands`                               |
| `language_name`        | Zeigt den Sprachnamen zu einem Code                                                    | `nl` → `Dutch`                                     |

## Datums- und Zeitangaben

Datumsformate verwenden Standard-Datumstoken wie `d` (Tag), `m` (Monat), `Y` (Jahr), `H` (Stunde) und `i` (Minute).

| Modifier                             | Was er macht                         | Beispiel                                  |
| ------------------------------------ | ------------------------------------ | ----------------------------------------- |
| `format_date: format`                | Formatiert ein Datum                 | `format_date: "d-m-Y"` → `14-03-2026`     |
| `format_relative_date`               | Zeigt das Datum relativ zu jetzt     | `4 months ago`                            |
| `date_add: interval`                 | Fügt einem Datum Zeit hinzu          | `date_add: "3 days"`                      |
| `date_subtract: interval`            | Subtrahiert Zeit von einem Datum     | `date_subtract: "1 week"`                 |
| `age`                                | Volle Jahre seit dem Datum           | Geburtsdatum → `34`                       |
| `diff_in_years: to`                  | Jahre zwischen zwei Daten            |                                           |
| `diff_in_months: to`                 | Monate zwischen zwei Daten           |                                           |
| `diff_in_weeks: to`                  | Wochen zwischen zwei Daten           |                                           |
| `diff_in_days: to`                   | Tage zwischen zwei Daten             |                                           |
| `diff_in_hours: to`                  | Stunden zwischen zwei Daten          |                                           |
| `diff_in_minutes: to`                | Minuten zwischen zwei Daten          |                                           |
| `diff_in_seconds: to`                | Sekunden zwischen zwei Daten         |                                           |
| `start_of: unit`                     | Anfang des Tages, der Woche, des Monats usw. | `start_of: "month"` → Erster des Monats |
| `end_of: unit`                       | Ende des Tages, der Woche, des Monats usw. | `end_of: "month"` → Letzter des Monats    |
| `is_past`                            | True, wenn das Datum in der Vergangenheit liegt |                                    |
| `is_future`                          | True, wenn das Datum in der Zukunft liegt |                                       |
| `is_today`                           | True, wenn das Datum heute ist       |                                           |
| `is_weekend`                         | True, wenn das Datum ein Wochenende ist |                                        |
| `day_name`                           | Name des Wochentags                  | `Thursday`                                |
| `month_name`                         | Name des Monats                      | `January`                                 |
| `quarter`                            | Quartal des Jahres (1–4)             | `1`                                       |
| `week` (`week_of_year`)              | Kalenderwoche des Jahres             | `3`                                       |
| `add_business_days: n`               | Fügt Tage hinzu und überspringt Wochenenden |                                    |
| `to_timezone: zone` (`tz`)           | Konvertiert in eine Zeitzone         | `to_timezone: "Europe/Amsterdam"`         |
| `to_timestamp` (`unix`, `timestamp`) | Das Datum als numerischer Timestamp  |                                           |
| `iso8601` (`iso_date`)               | Das Datum im standardisierten ISO-8601-Format | `2026-03-14T09:00:00+00:00`      |

Die Einheiten für `start_of` und `end_of` sind `year`, `quarter`, `month`, `week`, `day`, `hour`, `minute` und `second`.

## Bedingungen und Logik

Bedingungen geben true oder false zurück. Paaren Sie sie mit `boolean_label`, um ein freundliches Label anzuzeigen, oder kombinieren Sie sie mit `and`, `or` und `not`.

| Modifier                           | Was er macht                            | Beispiel                                               |
| ---------------------------------- | --------------------------------------- | ------------------------------------------------------ |
| `equals: value` (`eq`)             | True, wenn gleich                       | `{{ stage \| equals: "offer" }}`                       |
| `not_equals: value` (`ne`, `neq`)  | True, wenn nicht gleich                 |                                                        |
| `greater_than: n` (`gt`)           | True, wenn größer als                   |                                                        |
| `less_than: n` (`lt`)              | True, wenn kleiner als                  |                                                        |
| `greater_than_or_equal: n` (`gte`) | True, wenn größer oder gleich           |                                                        |
| `less_than_or_equal: n` (`lte`)    | True, wenn kleiner oder gleich          |                                                        |
| `between: min, max`                | True, wenn innerhalb des Bereichs       | `between: 0, 10`                                       |
| `is_empty` (`empty`)               | True, wenn das Feld leer ist            |                                                        |
| `is_not_empty` (`not_empty`)       | True, wenn das Feld einen Wert hat      |                                                        |
| `contains: text`                   | True, wenn es den Text enthält          |                                                        |
| `does_not_contain: text`           | True, wenn es den Text nicht enthält    |                                                        |
| `starts_with: text`                | True, wenn es mit dem Text beginnt      |                                                        |
| `ends_with: text`                  | True, wenn es mit dem Text endet        |                                                        |
| `does_not_start_with: text`        | True, wenn es nicht mit dem Text beginnt |                                                       |
| `does_not_end_with: text`          | True, wenn es nicht mit dem Text endet  |                                                        |
| `one_of: a, b, …` (`in`)           | True, wenn es zu einem Wert passt       | `one_of: "offer", "hired"`                             |
| `not_one_of: a, b, …` (`not_in`)   | True, wenn es zu keinem passt           |                                                        |
| `and: (condition)`                 | True, wenn beide Seiten wahr sind       | `gt: 80 \| and: (stage \| equals: "offer")`            |
| `or: (condition)`                  | True, wenn eine Seite wahr ist          |                                                        |
| `not`                              | Kehrt true und false um                 | `equals: "hired" \| not`                               |
| `boolean_label: yes, no`           | Zeigt ein Label für true/false          | `gt: 80 \| boolean_label: "Strong", "Weak"` → `Strong` |

Eine Bedingung kombiniert mit einem Label ist ein häufiges Muster:

```
{{ score | greater_than: 80 | boolean_label: "Strong candidate", "Needs review" }}
```
