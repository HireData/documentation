_Bienvenue dans notre guide concis sur la configuration d'une localisation pour votre entreprise dans HireData. Nous avons créé une intégration avec Google Maps qui vous permet d'ajouter une ou plusieurs localisations pour votre entreprise, mais pour l'utiliser, votre société doit être répertoriée sur [Google Business](https://www.google.com/intl/en_uk/business/), donc si vous n'avez pas encore enregistré votre profil, veuillez le faire avant de poursuivre ce guide._

## Étape 1 : Accéder au menu Locations

Connectez-vous à votre profil HireData, rendez-vous dans le menu des paramètres situé au-dessus de votre nom et ouvrez l'onglet [Locations].

![Menu Locations](/images/intercom/023fe782ebe4.png)

## Étape 2 : Ajouter une nouvelle localisation

_2.1. Cliquez sur le bouton [New Location] :_

![Bouton New Location](/images/intercom/a3c79bac597d.png)

_2.2. Nommez la localisation :_

![Champ Name Location](/images/intercom/c9a6c4d0f2fb.png)

_2.3. Sélectionnez sa localisation parmi les résultats :_

![Résultats de localisation](/images/intercom/5a3a6ff369f6.png)

_2.4. Cliquez sur le bouton [Create location]_

![Bouton Create Location](/images/intercom/19e3c341ff15.png)

_2.5. Vérifiez la localisation_

![Vue d'ensemble de la localisation](/images/intercom/c00e5b483b13.png)

## Astuce de pro

Une fois votre localisation ajoutée à votre compte HireData, vous obtiendrez automatiquement une URL Share Google Review, que vous pouvez envoyer à d'autres pour collecter des avis Google natifs. Ce type d'avis fonctionne toujours mieux grâce à l'algorithme de Google.

![Menu déroulant avec le bouton Share Google Review URL](/images/intercom/844129a640b5.png)

Enfin, vous pouvez également [connecter votre marque à la localisation](/fr/settings/brands). Ainsi, la localisation de votre entreprise sera automatiquement affichée dans la communication que vous utilisez, par ex. les e-mails, là où c'est nécessaire.
