_Bienvenue dans notre guide détaillé sur l'ajout automatique d'une activité ou d'un élément dans Vincere avec HireData. Ce tutoriel se veut simple et efficace, pour vous aider à exploiter tout le potentiel des automatisations Vincere par HireData._

Avant de commencer à configurer cette tâche automatisée, assurez-vous d'avoir [configuré la connexion avec Vincere](/fr/apps/vincere/connect-with-vincere) et de vous être familiarisé avec [la manière de créer une automatisation Vincere](/fr/apps/vincere/create-a-vincere-automation-in-hiredata) dans HireData. Passons maintenant à la configuration de cette tâche automatisée.

## Étape 1 : Ajouter une tâche à votre automatisation

### 1.1. Dans votre automatisation, cliquez sur le [+] pour ouvrir le Building Block Panel

![Ajouter un Building Block](/images/intercom/70816127-image.png)

### 1.2. Cliquez sur [Add] dans le Building Block Panel

![Building Block Panel](/images/intercom/70819499-image.png)

### 1.3. Choisissez Activity or Item Task

![Tâche Activity or Item](/images/intercom/70819730-image.png)

## Étape 2 : Choisir le type d'activité ou d'élément

Voici les types actuellement pris en charge parmi lesquels vous pouvez choisir :

- Candidate
- Candidate URL Document
- Candidate Note
- Company
- Contact
- Job
- Meeting

_Sélectionnez le type d'activité ou d'élément que vous souhaitez ajouter ou consigner dans Vincere :_

![Types d'activités ou d'éléments Vincere](/images/intercom/70821327-image.png)

## Étape 3 : Remplir les champs de l'élément

Vous pouvez maintenant compléter les champs de l'activité ou de l'élément. Vous avez la possibilité d'incorporer des variables en cliquant sur l'icône située en haut à droite. Des données à la fois statiques et dynamiques peuvent être utilisées pour remplir la tâche. N'oubliez pas de cliquer sur [Save Task] une fois terminé.

_Cliquez sur les accolades pour le menu Variables :_

![](/images/intercom/70822149-image.png)
