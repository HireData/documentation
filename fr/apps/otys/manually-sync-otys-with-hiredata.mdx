_Bienvenue dans notre manuel sur la synchronisation de votre compte OTYS avec HireData. Veuillez suivre les instructions ci-dessous pour synchroniser manuellement les objets, les statuts et les utilisateurs. Ainsi, vous pouvez utiliser un statut candidat, un champ personnalisé ou une étape de match que vous venez de créer ou de mettre à jour dans OTYS, sans attendre la prochaine synchronisation quotidienne entre vos comptes._

Pour cela, vous devez d'abord [connecter votre compte OTYS](/fr/apps/otys/connect-with-otys) à HireData.

Nous planifierons automatiquement une synchronisation pour importer les données des utilisateurs d'OTYS vers HireData une fois la connexion établie. Cela est nécessaire pour que les utilisateurs puissent envoyer automatiquement différents types de communications via notre plateforme. Nous planifions ensuite des resynchronisations quotidiennes au cas où vous apporteriez des modifications à votre profil OTYS.

Cependant, vous pouvez également synchroniser manuellement lorsque, par exemple, vous souhaitez voir immédiatement dans votre profil HireData les modifications que vous avez apportées à votre environnement Vincere. Voici comment procéder :

## Étape 1 : Accédez à la page Settings

Connectez-vous à votre compte HireData et ouvrez la page Settings via le bouton [Settings] situé au-dessus de votre nom.

![Bouton Settings](/images/intercom/49935101-image.png)

## Étape 2 : Ouvrez le menu Apps

![Page Apps dans HireData](/images/intercom/49935163-Screenshot-2025-05-16-at-11_18_28.png)

## Étape 3 : Sélectionnez l'app OTYS

Vous pouvez ouvrir l'app Vincere soit en cliquant sur le nom, soit sur le bouton [View app].

![Bouton View app](/images/intercom/49935188-Screenshot-2025-05-16-at-11_18_28-copy.png)

## Étape 4 : Synchronisez les données

Cliquez sur le bouton [Sync] situé dans le menu déroulant de la connexion de l'entreprise.

![Bouton Sync](/images/intercom/49935249-Screenshot-2025-05-16-at-11_25_45.png)

Après avoir cliqué dessus, vous verrez la progression de la synchronisation.

![Progression de la synchronisation](/images/intercom/49935294-Screenshot-2025-05-16-at-11_25_53.png)

Enfin, lorsque la synchronisation s'est terminée avec succès, vous devriez recevoir ce message en bas de la Data Sync Timeline :

![Data Sync Timeline](/images/intercom/49935331-Screenshot-2025-05-16-at-11_26_21.png)
