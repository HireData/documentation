_Bienvenue dans notre guide détaillé sur la connexion de votre compte OTYS à HireData. Ce tutoriel se veut simple et efficace, pour vous aider à exploiter tout le potentiel d'OTYS dans vos workflows HireData. Pour un accompagnement visuel, n'oubliez pas de regarder notre vidéo étape par étape._

## **Étape 1 : Accéder à l'app OTYS**

Commencez par vous rendre dans la [section des apps](https://app.hiredata.com/apps) de HireData. Vous y trouverez l'option pour [intégrer OTYS](https://app.hiredata.com/apps/otys). Cette étape initiale est simple mais fondamentale pour le processus d'intégration.

## **Étape 2 : Créer une nouvelle connexion**

Ensuite, dans HireData, sélectionnez l'app OTYS et choisissez d'ajouter une [+ New Connection]. Ce processus est conçu pour être direct, vous permettant d'intégrer les deux plateformes sans effort.

_Cliquez sur [+ New Connection] et commencez à connecter votre Carerix_

![](/images/intercom/3a4e37571212.png)

## **Étape 3 : Ajouter votre nom de compte OTYS et votre clé API**

Pour une intégration réussie, deux informations essentielles sont requises : votre nom de compte OTYS et la clé API. Si vous n'êtes pas sûr de ces détails, contactez votre représentant OTYS pour obtenir de l'aide.

_Fournissez votre nom de compte et votre clé API pour configurer une nouvelle connexion_
