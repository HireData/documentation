_Bienvenue dans notre manuel sur la synchronisation de votre compte Carerix avec HireData. Veuillez suivre les instructions ci-dessous pour synchroniser manuellement les objets, les statuts et les utilisateurs. Ainsi, vous pouvez utiliser un statut candidat, un champ personnalisé ou une étape de match que vous venez de créer ou de mettre à jour dans Carerix, sans attendre la synchronisation quotidienne entre vos comptes._

Pour cela, vous devez d'abord [connecter votre compte Carerix](/fr/apps/carerix/connect-with-carerix) à HireData.

Nous planifierons automatiquement une synchronisation pour importer les données des utilisateurs de Carerix vers HireData une fois la connexion établie. Cela est nécessaire pour que les utilisateurs puissent envoyer automatiquement différents types de communications via notre plateforme. Nous planifions ensuite des resynchronisations quotidiennes au cas où vous apporteriez des modifications à votre profil Carerix.

Cependant, vous pouvez également synchroniser manuellement lorsque, par exemple, vous souhaitez voir immédiatement dans votre profil HireData les modifications que vous avez apportées à votre environnement Carerix. Voici comment procéder :

## Étape 1 : Accédez à la page Settings

Connectez-vous à votre compte HireData et ouvrez la page Settings via le bouton [Settings] situé au-dessus de votre nom.

![Bouton Settings](/images/intercom/3cde4c7febdd.png)

## Étape 2 : Ouvrez le menu Apps

![Menu Apps](/images/intercom/0418411d95ee.png)

## Étape 3 : Sélectionnez l'app Carerix

Vous pouvez ouvrir l'app Carerix soit en cliquant sur le nom, soit sur le bouton [View app].

![Bouton View app](/images/intercom/5c8a51ef1364.png)

## Étape 4 : Synchronisez les données

Cliquez sur le bouton [Sync] (les deux flèches en mouvement circulaire) situé à la fin de la connexion de l'entreprise. Après avoir cliqué dessus, vous verrez la progression de la synchronisation.

![](/images/intercom/a7a4d05ef245.png)

## Astuce de pro

Lorsque vous cliquez sur le bouton [Timeline] (le premier des quatre boutons), vous pouvez voir ce qui a été synchronisé.

![Timeline de la synchronisation des données](/images/intercom/31cd3f5e5d7c.png)
