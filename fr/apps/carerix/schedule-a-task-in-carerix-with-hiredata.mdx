_Bienvenue dans notre guide complet sur la création d'une « Schedule task » dans Carerix avec HireData. Suivez les instructions ci-dessous pour apprendre le processus afin de pouvoir le gérer vous-même. À la fin, vous trouverez également un tutoriel vidéo pour un soutien supplémentaire._

Vous devriez déjà avoir créé une automatisation pour Carerix — découvrez comment faire [ici](/fr/apps/carerix/create-a-carerix-automation-in-hiredata).

## Étape 1 : Créer une tâche

Cliquez sur [+] pour ouvrir le menu Building Blocks, puis sur [Task] et enfin sélectionnez l'app [Carerix]. Vous verrez toutes les tâches automatisées que nous prenons actuellement en charge via Carerix :

- Add Activity or Item
- Update Field(s)
- Schedule task

Sélectionnez la tâche que vous souhaitez créer.

![Menu Building Blocks avec les tâches automatisées Carerix](/images/intercom/244c3d20ce45.png)

## Étape 2 : La configurer

Dans ce guide, nous nous concentrerons sur la création d'une « Schedule task ». Après avoir cliqué sur le bouton [Schedule Task], vous devez remplir quelques champs.

![Paramètres de la tâche](/images/intercom/75885f020ae2.png)

**_2.1. Choisir le propriétaire de la tâche_**

La première chose à faire après avoir créé la tâche est de vous assurer que le propriétaire est le même que celui du match. Pour ce faire, sélectionnez [ID] dans la case [Match Owner] et [Match: Owner (User)] dans la case [With] ensuite. Ainsi, la tâche sera attribuée à cette personne.

![Paramétrage de la correspondance du propriétaire](/images/intercom/720df4ec3271.png)

**_2.2. Définir une date de début et une date d'échéance_**

Selon le moment où vous souhaitez que la tâche commence et soit terminée, vous pouvez définir une [Start date] ainsi qu'une [Due Date].

![Options de date de début et d'échéance](/images/intercom/898d7cee1844.png)

**_2.3. Inclure un sujet_**

Dans la zone [Subject], vous pouvez ajouter des variables. Il y a deux façons de le faire. La première consiste à ouvrir les variables sur la droite via le bouton **\{\}**. Puis tapez n'importe quel mot-clé de la variable pour la trouver plus vite, et cliquez simplement dessus pour la copier.

![Liste des variables](/images/intercom/16948b32fa7d.png)

L'autre moyen est de taper « \{\{ » suivi du nom de la variable dans le champ sujet.

![Trouver une variable dans le champ sujet](/images/intercom/9520268e2ce8.png)

Par exemple, si vous voulez vérifier si le candidat est satisfait, vous devez taper « Check if \{\{CandidateFirstName is happy ».

![Exemple de sujet](/images/intercom/35af7e599608.png)

**_2.4. Rédiger une description_**

Rédigez une brève description de la tâche — elle ne doit pas dépasser une phrase.

![Exemple de description](/images/intercom/2849eab1e7a5.png)

**_2.5. Sélectionner le statut de la tâche_**

Il peut être :

- To do
- In progress
- Done
- Screening accepted
- Screening declined

![Options de statut de la tâche](/images/intercom/d0e5894aa343.png)

**_2.6. Ajouter une date d'expiration ou un délai_**

Ceci est facultatif, donc si vous ne souhaitez pas les ajouter, vous pouvez ignorer cette partie.

![Paramètres de date d'expiration et de délai](/images/intercom/b17544b14cd9.png)

**_2.6.1. Définir une date d'expiration pour la tâche_**

![Options de date d'expiration](/images/intercom/17be4d5d7a8a.png)

**_2.6.2. Définir un délai pour la tâche_**

![Options de délai](/images/intercom/93b6beee5bea.png)

**_2.7. Associer le candidat_**

Assurez-vous d'associer le bon candidat à cette tâche afin qu'elle corresponde également au dossier du candidat dans Carerix.

![Champ Match Candidate](/images/intercom/d9a4c4b18811.png)

**_2.8. Enregistrer la tâche_**

Cliquez sur [Save Task] en bas du menu Building Blocks.

![Bouton Save Task](/images/intercom/a468ea5bb9fc.png)

## Étape 3 : Vérifier la tâche

Vérifiez le récapitulatif de la tâche dans l'automatisation.
