_Bienvenue dans notre guide détaillé sur la connexion de votre compte Recruitee à HireData. Ce tutoriel se veut simple et efficace, pour vous aider à exploiter tout le potentiel de Recruitee dans vos automatisations HireData. Pour un accompagnement visuel, n'oubliez pas de regarder notre **vidéo Connect with Recruitee** à la fin._

## Étape 1 : Accéder à l'app Recruitee

Commencez par vous rendre dans la [section Apps](https://app.hiredata.com/settings/apps) de HireData. Vous la trouverez sous Settings &gt; Apps. Vous y verrez l'option pour [intégrer Recruitee.](https://app.hiredata.com/apps/recruitee) Cette étape initiale est simple mais essentielle pour le processus d'intégration.

![](/images/intercom/65643328-Schermafbeelding-2B2025-09-19-2B100347.png)

## Étape 2 : Créer une connexion avec Recruitee

Ouvrez l'app et cliquez sur **Connect to Recruitee** pour démarrer le processus d'intégration. Cela permettra à HireData de se connecter à votre compte Recruitee.

![](/images/intercom/65646660-Schermafbeelding-2B2025-09-19-2B102104.png)

### 2.1. Créer un jeton API

Cliquez sur '**here'** pour créer un nouveau jeton API dans Recruitee.

Assurez-vous que votre utilisateur dispose de ces rôles :

- Basic permissions → Collaborate on candidates, Join hiring teams
- Jobs → View all existing jobs data
- Hires → View hired candidates
- Add-ons → Manage API tokens

Copiez maintenant le jeton API généré et collez-le ci-dessous pour connecter Recruitee à HireData.

![](/images/intercom/65646659-image.png)

### 2.2. Copier votre Recruitee Company ID

Sous **Current company details in Recruitee**, vous verrez votre **Company ID**. Copiez simplement cet ID et collez-le dans le champ HireData.

![](/images/intercom/65646661-image.png)

Une fois rempli, cliquez sur [Continue] pour finaliser la connexion avec Recruitee.

![](/images/intercom/66842172-image.png)

## Étape 3 : Recruitee connecté !

Après avoir cliqué sur [Continue], vous avez terminé la configuration de la connexion entre Recruitee et HireData. S'il s'agit de la première fois que vous configurez cette connexion, HireData peut avoir besoin de quelques minutes pour synchroniser les données depuis votre base Recruitee. Ensuite, vous êtes prêt à [configurer votre première automatisation Recruitee](/fr/apps/recruitee/create-a-recruitee-automation-in-hiredata) dans HireData.

![](/images/intercom/66841878-image.png)
