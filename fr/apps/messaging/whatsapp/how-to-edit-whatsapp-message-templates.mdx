_Les modèles de messages WhatsApp dans HireData vous permettent de gérer et de mettre à jour les messages utilisés dans les automatisations. Cet article vous guide pour modifier les modèles de messages existants directement dans HireData._

<Steps>
  <Step title="Ouvrir le modèle à modifier">
    Rendez-vous dans vos **Settings** et ouvrez la page **Messages**. Vous y verrez la liste de vos modèles WhatsApp créés dans HireData ou synchronisés avec votre WhatsApp Business Manager.

    Sélectionnez le modèle que vous souhaitez modifier dans la liste.
  </Step>
  <Step title="Modifier votre modèle de message">
    Mettez à jour le contenu du message selon vos besoins. Utilisez l'écran d'aperçu à droite pour voir comment votre message apparaîtra aux destinataires avant d'enregistrer.

    ![L'éditeur de modèle de message avec un aperçu en direct à droite](/images/intercom/146959675-Screenshot-2026-05-29-at-10_28_58.png)

    Une fois satisfait de vos modifications, cliquez sur **Save** dans le coin supérieur droit pour les appliquer.

    ![Le bouton Save dans l'éditeur de modèle](/images/intercom/146960062-Screenshot-2026-05-29-at-10_30_25.png)
  </Step>
</Steps>

<Warning>
  Chaque modèle ne peut être modifié qu'une seule fois toutes les **24 heures**. Planifiez soigneusement vos changements avant d'enregistrer, car vous ne pourrez pas apporter d'autres modifications avant le lendemain.

  Si vous essayez de le modifier plusieurs fois dans ce délai, vous verrez un message d'alerte sur le modèle.
</Warning>

Pour en savoir plus sur le fonctionnement des modèles et leur approbation, consultez [Comment fonctionnent les modèles WhatsApp](/fr/apps/messaging/whatsapp/how-whatsapp-templates-work).
