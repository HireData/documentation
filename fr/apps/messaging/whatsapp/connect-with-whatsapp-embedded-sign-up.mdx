_Bienvenue dans notre guide sur la configuration de WhatsApp dans HireData. Suivez-nous pour découvrir chaque étape de la connexion des deux plateformes via l'inscription intégrée._

## Étape 1 : Choisir une méthode de connexion

Vous pouvez soit le faire depuis votre [Inbox] puis [WhatsApp], soit vous rendre sur la [page Settings], puis [Apps], et ouvrir [WhatsApp].

![Création ou sélection d'un compte WhatsApp Business](/images/intercom/23699831-Screenshot-2B2024-10-23-2B160600.jpg)

Deux options vous sont proposées pour connecter votre compte WhatsApp à HireData :

- **Embedded Sign-Up** : cette option vous permet d'utiliser la fonctionnalité WhatsApp Embedded Sign-Up pour une configuration plus facile.
- **[System User](/fr/apps/messaging/whatsapp/connect-with-whatsapp-embedded-sign-up)** : pour connecter votre compte WhatsApp via cette option, vous aurez besoin du **WhatsApp Business Account Id** et de l'**Access Token** (clé API).

![Embedded Sign-Up vs System User](/images/intercom/23699845-Screenshot-2B2024-10-23-2B160645.jpg)

## Étape 2 : Accorder à HireData l'accès à WhatsApp Business

### 2.1. Une fois que vous avez sélectionné une méthode de connexion, vous serez redirigé vers Facebook, où vous devrez vous connecter à votre compte Facebook

![Page de connexion Facebook](/images/intercom/23700795-1-2Bstep.jpg)

### 2.2. Vous serez ensuite redirigé vers la page des autorisations, où il vous sera demandé de connecter votre compte WhatsApp Business à HireData

En continuant, vous devrez accepter les [Conditions de service WhatsApp Business](https://www.whatsapp.com/legal/business-terms?fbclid=IwZXh0bgNhZW0CMTAAAR0O-c4JvRz5u9AKYZq_4QBZjltjfxac5AoPTh3E31K3Ho2TtD8l-JP4DiA_aem_-GN92crMv9g93Rp_HF4Q-w) et les [Conditions de service Meta](https://business.facebook.com/legal/terms).

![Page d'autorisations du compte WhatsApp Business](/images/intercom/23700821-Screenshot-2B2024-10-23-2B154735.jpg)

## Étape 3 : Renseigner les informations de votre entreprise

Vous devrez remplir les champs d'informations sur votre entreprise, notamment :

- **Business Portfolio** : choisissez un portefeuille existant ou créez-en un nouveau.
- **Business Name** : saisissez le nom de votre entreprise.
- **Business Website or Profile Page** : le cas échéant, saisissez l'URL du site web ou du profil sur les réseaux sociaux de votre entreprise.
- **Country** : sélectionnez le pays dans lequel votre entreprise est basée.
- **Address (facultatif) :** vous pouvez également ajouter l'adresse de votre entreprise si vous le souhaitez.

Une fois terminé, cliquez sur [Next] pour continuer.

![Page des informations de l'entreprise](/images/intercom/23701899-new.jpg)

## Étape 4 : Créer ou sélectionner votre compte WhatsApp Business

Si vous disposez déjà d'un compte WhatsApp Business, vous pourrez le sélectionner à ce stade, mais si vous n'en avez pas, il vous sera demandé d'en créer un nouveau.

![Création ou sélection d'un compte WhatsApp Business](/images/intercom/23702357-Screenshot-2B2024-10-23-2B154956.jpg)

### 4.1. Créer un nouveau compte WhatsApp Business

Remplissez les champs requis, notamment :

- **WhatsApp Business Account Name** : saisissez le nom de votre compte WhatsApp Business. Il ne doit pas dépasser 255 caractères.
- **WhatsApp Business Display Name** : assurez-vous que le nom d'affichage correspond au nom de votre entreprise et respecte les [directives de nom d'affichage de WhatsApp](https://www.facebook.com/business/help/757569725593362).
- **Category** : sélectionnez la catégorie d'entreprise qui décrit le mieux votre activité.

Cliquez sur [Next] pour continuer.

![Création d'un nouveau compte WhatsApp Business](/images/intercom/23702845-Screenshot-2B2024-10-23-2B155036.jpg)

## Étape 5 : Ajouter un numéro de téléphone pour WhatsApp

### 5.1. Saisissez le numéro de téléphone qui sera lié à votre compte WhatsApp Business

- **Phone Number** : sélectionnez le bon indicatif pays et saisissez votre numéro de téléphone.

Si le numéro de téléphone que vous essayez d'utiliser est déjà lié à WhatsApp, vous devrez soit [acheter un nouveau numéro de téléphone](/fr/apps/messaging/whatsapp/buy-a-phone-number-from-numero-for-your-whatsapp-account), soit supprimer votre compte WhatsApp et réessayer. Rappelez-vous que si vous faites cela, votre historique de chat sera supprimé.

- **Verification** : choisissez comment vous souhaitez vérifier le numéro de téléphone, via un SMS ou un appel téléphonique.

![Ajout d'un numéro de téléphone pour WhatsApp](/images/intercom/23703206-Screenshot-2B2024-10-23-2B155239.jpg)

### 5.2. Vérifiez votre numéro de téléphone en saisissant le code à 6 chiffres que vous recevrez

![Page de vérification](/images/intercom/23798726-Untitled-design-2810-29.png)

## Étape 6 : WhatsApp connecté

### 6.1. Vérifiez la demande d'accès de HireData

![Page de demande d'accès](/images/intercom/23795609-Screenshot-2B2024-10-24-2B131229.jpg)

### 6.2. Message de confirmation

Si vous avez tout configuré correctement, vous verrez ce message à la fin, et vous serez prêt à utiliser HireData avec WhatsApp !

![Page de confirmation](/images/intercom/23795636-Screenshot-2B2024-10-24-2B133352.jpg)

### 6.3. Vérifier la connexion entre WhatsApp et HireData

Rendez-vous simplement dans [Settings] &gt; [Apps] &gt; [WhatsApp] et vérifiez si la connexion entre HireData et WhatsApp est réussie.

![Connexion WhatsApp dans HireData](/images/intercom/23798221-Untitled-2Bdesign-2B-288-29.png)

Maintenant que vous connaissez toutes les étapes, visitez notre [page d'inscription WhatsApp](https://app.hiredata.com/inbox/whatsapp) pour commencer à configurer votre connexion WhatsApp. Si vous rencontrez des problèmes, vous pouvez également essayer de connecter votre compte à HireData en utilisant une [connexion via System User](/fr/apps/messaging/whatsapp/connect-with-whatsapp-embedded-sign-up).
