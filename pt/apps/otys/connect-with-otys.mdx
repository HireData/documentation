_Bem-vindo ao nosso guia detalhado sobre como ligar a sua conta OTYS à HireData. Este tutorial foi concebido para ser prático e eficiente, ajudando-o a tirar o máximo partido do OTYS nos seus fluxos de trabalho HireData. Para um guia visual, não deixe de ver o nosso vídeo passo a passo._

## **Passo 1: Navegar para a app OTYS**

Comece por navegar até à [secção de apps](https://app.hiredata.com/apps) dentro da HireData. Aqui, encontrará a opção para [integrar com o OTYS](https://app.hiredata.com/apps/otys). Este primeiro passo é simples mas fundamental para o processo de integração.

## **Passo 2: Criar uma nova ligação**

De seguida, dentro da HireData, selecione a app OTYS e opte por adicionar uma [+ New Connection]. Este processo foi concebido para ser simples, permitindo-lhe integrar as duas plataformas sem esforço.

_Clique em [+ New Connection] e comece a ligar o seu Carerix_

![](/images/intercom/3a4e37571212.png)

## **Passo 3: Adicionar o nome da conta OTYS e API key**

Para uma integração bem-sucedida, são necessárias duas informações críticas: o nome da sua conta OTYS e a API key. Se tiver dúvidas sobre estes detalhes, contacte o seu representante OTYS para assistência.

_Forneça o nome da conta e a API key para configurar uma nova ligação_
