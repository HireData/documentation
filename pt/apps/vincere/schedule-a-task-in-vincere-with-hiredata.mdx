_A ação Vincere Scheduled Task permite-lhe criar automaticamente scheduled tasks no Vincere sempre que ocorre um evento de trigger específico num fluxo de trabalho de automação da HireData. Isto é útil para automatizar lembretes e atividades de seguimento, garantindo que nenhuma tarefa é esquecida. Acompanhe-nos enquanto demonstramos como utilizar esta tarefa._

## Passo 1: Criar ou editar uma automação

Na sua conta HireData, navegue até à secção **Automations** e crie um novo fluxo de trabalho ou abra um existente clicando em **+ New Automation**.

![Adding new automation in HireData](/images/intercom/70618331-image.png)

## Passo 2: Adicionar a ação

### 2.1. Clique em **Blank Automation** para iniciar uma nova automação a partir do zero

Selecionar Blank Automation criará uma nova automação

![Start a blank automation](/images/intercom/73792029-image.png)

### 2.2. Clique no primeiro Building Block/Trigger para abrir o App Panel e selecione **Vincere**

Ao clicar no primeiro Building Block, poderá escolher a que App associar

![First step to creating a Trigger by selecting the brand](/images/intercom/72641866-image.png)

### 2.3. Selecionar o Trigger para a automação

Siga os passos de configuração para escolher como pretende que esta automação seja acionada e clique em **Save**.

Selecionar o valor Object para a app

![Trigger Setup Choosing Object](/images/intercom/72645601-image.png)

Escolher o timing ao qual pretende que o trigger obedeça

![Trigger Setup Timing](/images/intercom/72645827-image.png)

Verificar se está tudo correto e guardar o trigger

![Overview of Trigger](/images/intercom/72646002-image.png)

### 2.4. Clique no 2º Building Block para abrir o painel de ações e clique em **Add** para criar uma nova tarefa

![](/images/intercom/72648002-image.png)

### 2.5. Navegue até ver o logótipo do Vincere junto ao texto **Task**

![Add Task](/images/intercom/72648586-image.png)

## Passo 3: Preencher os detalhes da tarefa

- **Type:** Escolha o Item Type ao qual quer que a tarefa corresponda. Pode ser:
  - Candidate – Uma tarefa relacionada com o Job Seeker
  - Contact – Uma tarefa associada ao Client Contact
  - Job – Uma tarefa relacionada com um Job Order ou vaga
  - Personal – Uma tarefa para uso próprio, como lembretes ou tarefas administrativas
  - Project – Uma tarefa associada a um projeto ou campanha maior

![](/images/intercom/72656330-image.png)

- **Subject:** Introduza um assunto claro e descritivo para a tarefa (por exemplo, "Test project, local")
- **Description:** Forneça uma descrição detalhada da tarefa
- **Start/End Date:** Defina as datas de início e fim da tarefa. Também pode definir um atraso, se quiser
- **Reminders:** Adicione um lembrete para garantir que a tarefa não é esquecida
- **Assignee:** Atribua a tarefa ao utilizador relevante no Vincere
![Task configuration](/images/intercom/70620488-image.png)

## Passo 4: Guardar e testar

Clique em **Save Task** para finalizar a configuração da tarefa e [correr o fluxo de trabalho da automação para o testar](/pt/automations/how-to-test-an-automation).

## Passo 5: Confirmar a tarefa

- **HireData** – Navegue até **Task History** para verificar se foi criada.

![Tracking the task in HireData](/images/intercom/70620685-image.png)

- **Vincere** – Abra **Email View → Task Tab**.

![Tracking the task in Vincere](/images/intercom/70620741-image.png)
