_Bem-vindo ao nosso guia detalhado sobre como ligar a sua conta Vincere à HireData. Este tutorial foi concebido para ser prático e eficiente, ajudando-o a tirar o máximo partido do Vincere nas suas automações HireData. Para um guia visual, não deixe de ver o nosso vídeo passo a passo._

## **Passo 1: Navegar para a app Vincere**

Comece por navegar até à [secção de apps](https://app.hiredata.com/apps) dentro da HireData. Aqui, encontrará a opção para [integrar com o Vincere](https://app.hiredata.com/apps/vincere). Este primeiro passo é simples mas fundamental para o processo de integração.

## **Passo 2: Criar uma nova ligação**

Clique em [+ New Connection] e introduza os detalhes necessários do Vincere: o seu tenant, client ID e API key. Se ainda não tem client ID e API key, contacte o seu account manager na Vincere. Damos prioridade à sua segurança, garantindo que as suas credenciais são protegidas durante todo o processo de integração.

_2.1: Clique em [+ New Connection] e comece a ligar o seu Vincere_

![](/images/intercom/a4141223e671.png)

_2.2: Forneça o seu tenant, client ID e API key para configurar uma nova ligação_

![](/images/intercom/0690462fc694.png)

## **Passo 3: Iniciar sessão no Vincere**

Depois de configurar a sua ligação, ser-lhe-á pedido para iniciar sessão no Vincere. Certifique-se de que inicia sessão com um utilizador que tenha direitos de admin atribuídos.

_Inicie sessão no Vincere utilizando um utilizador com direitos de admin:_

![](/images/intercom/9017e6375db2.png)

## **Passo 4: Vincere ligado!**

Depois de iniciar sessão no seu ambiente Vincere e dar o seu consentimento à HireData, concluiu a configuração da ligação entre o Vincere e a HireData. Consulte o tutorial para um guia passo a passo completo sobre como configurar a ligação.
