_Bem-vindo ao nosso guia passo a passo sobre como atualizar um ou mais campos em diferentes objetos no Vincere com a HireData. Siga este guia simples mas útil para perceber como atualizar qualquer campo por conta própria._

Antes de atualizar os campos no objeto, já deverá ter criado uma [automação no Vincere](/pt/apps/vincere/create-a-vincere-automation-in-hiredata).

## Passo 1: Abrir o menu Building Blocks

Primeiro, inicie sessão na sua conta HireData e abra a automação que pretende editar. Depois clique no ícone de mais [+] no final da automação para abrir o menu Building Blocks do lado direito do ecrã. Depois de abrir o menu, selecione [Update].

![Building Blocks menu](/images/intercom/51931242-Screenshot-2025-06-02-at-09_16_11.png)

## Passo 2: Selecionar o objeto que pretende editar

Escolha o objeto que quer editar, que atualmente pode ser um destes:

- Candidate
- Company
- Contact
- Job
- Meeting
- Task

_Para este exemplo, selecionámos o candidato:_

![Selecting an object to be updated](/images/intercom/51931309-Screenshot-2025-06-02-at-09_19_18.png)

## Passo 3: Selecionar um identificador

Agora tem de selecionar um identificador, que será utilizado para encontrar o objeto que pretende atualizar.

![Selecting an object identifier](/images/intercom/51931462-Screenshot-2025-06-02-at-09_19_42.png)

Para este guia, vamos usar o [ID] como identificador e o [Candidate: ID] como o valor que queremos seguir. Por outras palavras, vamos usar o Candidate ID para identificar o candidato que queremos atualizar na automação.

**_Os valores apresentados serão diferentes consoante o objeto e o identificador selecionados._**

![Selecting object identifier](/images/intercom/51931552-Screenshot-2025-06-02-at-09_19_50.png)

## Passo 4: Atualizar os campos

Agora, pode selecionar todos os campos que pretende atualizar — pode selecionar quantos quiser, basta usar a barra de pesquisa para os encontrar.

![Selecting field to be updated](/images/intercom/51931636-Screenshot-2025-06-02-at-09_20_14.png)

Edite os campos e clique no botão [Save Task] no fundo da página.

![Saving the update fields task](/images/intercom/51931666-Screenshot-2025-06-02-at-09_22_04.png)

## Passo 5: Rever

Por último, reveja o pop-up da tarefa, que aparecerá no fim da automação, para garantir que tudo corre sem problemas.

![Update field task](/images/intercom/51931722-Screenshot-2025-06-02-at-09_22_36.png)
