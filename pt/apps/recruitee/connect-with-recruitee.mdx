_Bem-vindo ao nosso guia detalhado sobre como ligar a sua conta Recruitee à HireData. Este tutorial foi concebido para ser prático e eficiente, ajudando-o a tirar o máximo partido do Recruitee nas suas automações HireData. Para um guia visual, não deixe de ver o nosso vídeo passo a passo **Connect with Recruitee Video** no final._

## Passo 1: Navegar para a app Recruitee

Comece por navegar até à [secção Apps](https://app.hiredata.com/settings/apps) dentro da HireData. Pode encontrá-la em Settings &gt; Apps. Aqui, verá a opção para [integrar com o Recruitee.](https://app.hiredata.com/apps/recruitee) Este primeiro passo é simples mas essencial para o processo de integração.

![](/images/intercom/65643328-Schermafbeelding-2B2025-09-19-2B100347.png)

## Passo 2: Criar uma ligação com o Recruitee

Abra a app e clique em **Connect to Recruitee** para iniciar o processo de integração. Isto permitirá que a HireData se ligue à sua conta Recruitee.

![](/images/intercom/65646660-Schermafbeelding-2B2025-09-19-2B102104.png)

### 2.1. Criar API token

Clique em '**here'** para criar um novo API token no Recruitee.

Certifique-se de que o seu utilizador tem estas funções:

- Basic permissions → Collaborate on candidates, Join hiring teams
- Jobs → View all existing jobs data
- Hires → View hired candidates
- Add-ons → Manage API tokens

Agora copie o API token gerado e cole-o abaixo para ligar o Recruitee à HireData.

![](/images/intercom/65646659-image.png)

### 2.2. Copiar o seu Recruitee Company ID

Em **Current company details in Recruitee**, verá o seu **Company ID**. Basta copiar este ID e colá-lo no campo da HireData.

![](/images/intercom/65646661-image.png)

Uma vez preenchido, clique em [Continue] para concluir a ligação com o Recruitee.

![](/images/intercom/66842172-image.png)

## Passo 3: Recruitee ligado!

Depois de clicar em [Continue], concluiu a configuração da ligação entre o Recruitee e a HireData. Quando é a primeira vez que configura esta ligação, a HireData poderá necessitar de alguns minutos para sincronizar dados da sua base de dados Recruitee. Depois disso, está pronto para [configurar a sua primeira automação Recruitee](/pt/apps/recruitee/create-a-recruitee-automation-in-hiredata) na HireData.

![](/images/intercom/66841878-image.png)
