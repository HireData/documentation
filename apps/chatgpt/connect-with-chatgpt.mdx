_Welcome to our detailed guide on connecting your ChatGPT account with HireData. This tutorial is designed to be user-friendly and efficient, helping you to leverage the full potential of ChatGPT within your HireData automations._

<Accordion title="Step 1: Create a Secret API Key in OpenAI">

API keys allow you to authenticate your requests to the OpenAI API. As a project owner, you can create and manage keys for your projects, which you can use to connect to different platforms, including HireData.

### 1.1. Head to [OpenAI](https://platform.openai.com/account/api-keys) and open the API keys page

![API Keys page](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42105010/c7a7ed3afd727381256cb01211ae/Screenshot+2025-03-20+at+16_31_03.png?expires=1776771000&signature=c8b20897313283ace6a480c88d875d7f19db46188790ce17b769c94ffa9a878d&req=1NduxV36rTdk2hL085ZhoRFc2vk4RwM6YpKH9%2F1hvtUORifhjb%2Fq93cnLwoV%0AXQ%3D%3D%0A)

### 1.2. Click on [Create new secret key]

### 1.3. Specify:

- **Ownership**: Select "You" (personal access)
- **Name**: Give your key a descriptive name (e.g., "Test API Key")
- **Project**: Select the project this key will be associated with (e.g., "HireData")
- **Permissions**: Give us permission to "All" (full read and write access)

![Creating a secret key](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42105278/0b8b1d87557978f11a24c03fdfe2/Screenshot+2025-03-17+at+05_13_24.png?expires=1776771000&signature=0d16515b9ddd3fd366ab04fc27f890adfc3127690a5ff859b05072c8ad2ee743&req=1NduxV34qz9k2hL085Zhof2J%2BKhYiGUMaMywDYtYbPEn4tekyz2bdUL28uE2%0AXw%3D%3D%0A)

### 1.4. Click on [Create secret key] to generate your new API key

### 1.5. Save and store your key somewhere save

![Saving the API key](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42105472/c19a01633f20176db78733283cf8/Screenshot+2025-03-17+at+05_13_54.png?expires=1776771000&signature=b908386565fd0eae7b22f8e82bd65fba2e7d4ef3b7bf3a85d9c9a096a72d5fcf&req=1NduxV3%2BqzVk2hL085ZhoR3hno4vA2f0WS6mUWl6XajGdxZuXosySsS561LO%0Abw%3D%3D%0A)

</Accordion>

<Accordion title="Step 2: Creating a New Connection in HireData">

### 2.1. Navigate to the Setting page and open the Apps section

![Apps page](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42106746/4785e854fdaf28355c9ea6bd8cf9/Screenshot+2025-03-19+at+21_56_34.png?expires=1776771000&signature=5b6b16805fed27e796c05b0b242c651c6fc70e7e267efe15dfbcef096a5892b4&req=1NduxV79qDFk2hL085ZhoTeSFijVooNnTcmqSHebWDCRPQMy7tTUf%2FxUleQC%0Apw%3D%3D%0A)

### 2.2. Next, within HireData, select the ChatGPT app and opt to add a [+ Setup ChatGPT]

This process is designed to be straightforward, allowing you to integrate both platforms effortlessly.

![ChatGPT app](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42106810/fa4350ce1bde0af343ef9de74ffc/Screenshot+2025-03-19+at+21_56_46.png?expires=1776771000&signature=a6c9d24da9b889b0ea0a25f9af0c1a9f8b3ca25885e7c1462c05609055780fa0&req=1NduxV7yrTdk2hL085ZhobZ%2B0RIV9sTEcNd0XYvayBkNjalr7ttw0otLtctA%0Amw%3D%3D%0A)

<Accordion title="💡 Tips">

- Do not share your API key with others
- Never expose your API key in the browser or other client-side code
- Keep your key secure - anyone with your API key can make requests on your behalf
- OpenAI may automatically disable any API key that has been leaked publicly

By following these guidelines, you can securely manage API access to your OpenAI projects while maintaining proper security protocols.

</Accordion>
