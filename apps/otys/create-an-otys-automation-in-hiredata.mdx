_Welcome to our tutorial on setting up an OTYS automation within HireData. With this integration, you can automate tasks such as sending Candidate NPS surveys, thereby improving candidate experiences and streamlining your recruitment processes. Follow along as we demonstrate each step to set up this automation efficiently._

## Step 1: Setting up the Trigger

After you've [set up the connection with OTYS](/apps/otys/connect-with-otys), the first step in setting up an OTYS automation is to define the trigger. A trigger makes sure HireData "listens" to certain events occurring within OTYS. Don't forget to select all relationships you'll need later on, e.g. select a candidate when you want to send a survey to them in a further step within the automation. Here's how to do it:
