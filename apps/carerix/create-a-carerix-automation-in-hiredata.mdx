_Welcome to our tutorial on setting up a Carerix automation within HireData. With this integration, you can automate tasks such as sending Candidate NPS surveys, thereby improving candidate experiences and streamlining your recruitment processes. Follow along as we demonstrate each step to set up this automation efficiently._

<Accordion title="Step 1: Setting up the Trigger">

After you've [set up the connection with Carerix](https://help.hiredata.com/en/articles/157034-set-up-a-connection-with-carerix), the first step in setting up a Carerix automation is to define the trigger. A trigger makes sure HireData "listens" to certain events occurring within Carerix. Don't forget to select all relationships you'll need later on, e.g. select a candidate when you want to send a survey to them in a further step within the automation. Here's how to do it:

_Example Trigger: Match: Created or Updated_

_Select the related objects you'll need later on in the automation, e.g. a user you want to be the sender of a message and the recipient(s) of that message_

![](https://downloads.intercomcdn.eu/i/o/12542640/dc94f04335afe0dcaf03bb9b/Screenshot+2024-04-16+at+08_53_34.png?expires=1776771000&signature=d4d5bbe48f3b918b692b89e4ee60f10973c1bfe14e0a7d0c189e48cdafe597a9&req=0ddqwVr8qDdk2hL085ZhobPmwXvpJ5UM%2FY15O47SUe8tXdDKDhwQyjMjR1xs%0Aqw%3D%3D%0A)
