_Welcome to our step-by-step guide on how to update one or more fields in different objects in Carerix with HireData. Follow this simple but helpful guide to understand how to update any field yourself. At the end, you will also find a video tutorial for further guidance._

Before updating the fields in the object, you should have already created an [automation in Carerix](https://help.hiredata.com/en/articles/150808-create-a-carerix-automation-in-hiredata).

## Step 1: Open the building blocks menu

First, log in to your HireData account and open the automation you want to edit. Then click on the plus icon [+] at the end of the automation to open the Building Blocks menu. You will see it open up on the right side of the screen.

![Building Blocks menu](https://downloads.intercomcdn.eu/i/o/14255716/8e9a155d84afced1fb736fd1/Capture-2024-05-30-153101.png?expires=1776771000&signature=01dee2b8aeb66fe4ed9f823b2d26d28ce87f895ce251e30b30d06f6c2952f721&req=0dFtwF39rTFk2hL085ZhoborYCrNVWArDMaoN0JpuIlQzm5vMyctJjA1UcvS%0ArQ%3D%3D%0A)

## Step 2: Select the Object you want to edit

After you have opened the menu, select [Task], then [Carerix], and choose a connection. You will then see all of the tasks that we support. One of them is [Update Field(s)] Click on it.

![Carerix tasks in HireData](https://downloads.intercomcdn.eu/i/o/14478052/d6c5dcd13445c468f43379da/Capture-2024-05-30-153751.png?expires=1776771000&signature=e2201723256cec14d9c4f98ef3c347d70a0875be7aa87969c1f24ad8b6a8def9&req=0dFrwlD6qTVk2hL085ZhoTZZwukuev%2Ffqx%2BPtPpf4HuDAzHzwUD2I7qa9Jnw%0A0Q%3D%3D%0A)

At this point, you have to choose the object you want to edit, which can be one of these:

- Appointment
- Candidate
- Company
- Match
- Placement
- Publication
- User
- Vacancy

_For this example, we have selected the candidate:_

![](https://downloads.intercomcdn.eu/i/o/14477450/438620fa54970015469ef661/image.png?expires=1776771000&signature=bda73c880e65ff3dcdbcad1d5e99a8810e9184143644be7e3856ba47dd5676a8&req=0dFrwl%2F%2BqTdk2hL085Zhoa3sThyqrJ%2F9fPj%2FPbtbkN9slPlVphJRPAgqEeX%2B%0Auw%3D%3D%0A)

## Step 3: Select an Identifier

Now you have to select an identifier, which will be used to find the object in the automation. This could be either an ID or a Business/Private email address.

![Identifier options](https://downloads.intercomcdn.eu/i/o/14477493/7c3b7c6fa10f38fa4990f712/image.png?expires=1776771000&signature=053b95c302267be3a3c24f1426a5160f3cf3613cc75033412519c83d3ddfd66e&req=0dFrwl%2F%2BpTRk2hL085ZhodzBkIMfkF6d%2FJJW9Bk9BVe2EFGcxh1TZXKoc6jb%0Ajw%3D%3D%0A)

For this guide, we will use the [ID] as an identifier and the [Candidate: ID] as the value we want to track. In other words, we will use the Candidate ID to track the candidate we want to update in the automation.

**_The displayed values will be different depending on the selected object and identifier._**

![Value options](https://downloads.intercomcdn.eu/i/o/14477554/ea92281bd1ffc67a94006466/image.png?expires=1776771000&signature=e89578a3cfcc0f17215bf979568dfecc02d587109a7a2fc5077b0ff7412db9f5&req=0dFrwl%2F%2FqTNk2hL085ZhocH49OTKdd1WrHcDQYc7oCayF6%2F62xyEqQtnDpsJ%0ASQ%3D%3D%0A)

## Step 4: Update the fields

Now, you can select all the fields you want to update—you can select as many as you want, just use the search bar to find them.

![Choosing which fields to update](https://downloads.intercomcdn.eu/i/o/14256861/7aa276d38007d568dc5e5598/Capture-2024-05-30-155632.png?expires=1776771000&signature=6fdf58d943fc3dce191663d3dbfdfc1120a01fae8164a4d2bee978b3d7e1103e&req=0dFtwF7yqjZk2hL085ZhodDUU%2B1Xgnc8Pn2Q9DFZauJ05tP5hLcL0dKakWIe%0AOA%3D%3D%0A)

Edit the fields and click the [Save Task] button at the bottom of the page.

_This is an example of some of the fields you can update:_

![Example fields that can be updated](https://downloads.intercomcdn.eu/i/o/14256725/264233c422d5791949dbc46e/Capture-2024-05-30-155152.png?expires=1776771000&signature=f46dcd45b5ccedfa35bc9e4f0c90f686e168df085d315aa2b41711a0ed3caff4&req=0dFtwF79rjJk2hL085Zhobb2VTUzy6kQwWSEqo62%2FDgtQnhy4W8lrF7AfC2l%0ALg%3D%3D%0A)

## Step 5: Review

Last but not least, review the task pop-up, which will appear at the end of the automation, to ensure everything runs smoothly.

![Task summary](https://downloads.intercomcdn.eu/i/o/14256999/1306a57db4e39259990241ab/Capture-2024-05-30-155922.png?expires=1776771000&signature=f8a32d8eda4b09a89d4c29a4383454a0b341c1cfc14064bcf1d51e0aa627760f&req=0dFtwF7zpT5k2hL085Zhoe5Yhbcu%2F3wI%2Fq5wqS%2FnpBxL6KVRAx4fjcM1M%2BRz%0AMg%3D%3D%0A)
