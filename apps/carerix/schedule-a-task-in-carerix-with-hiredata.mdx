_Welcome to our comprehensive guide on how to create a "Schedule task" in Carerix with HireData. Follow the instructions below to learn the process so you can manage it yourself. At the end, you will also find a video tutorial for additional support._

You should already have created an automation for Carerix - see how to do this [here](https://help.hiredata.com/en/articles/150808-create-a-carerix-automation-in-hiredata).

<Accordion title="Step 1: Create a task">

Click the [+] to open the Building Blocks menu, then on [Task] and finally select the [Carerix] app. You will see all of the automated tasks that we currently support through Carerix:

- Add Activity or Item
- Update Field(s)
- Schedule task

Select the task you want to create.

![Building Blocks menu with Carerix automated tasks](https://downloads.intercomcdn.eu/i/o/14260144/f3f2c7fa22835fcef53f5830/Capture-2024-05-30-153751.png?expires=1776771000&signature=6f283d1ad3cdf3684ab69d4409e6d2f5e3684587919fcf3a163a8d90f82c9ada&req=0dFtw1j7qDNk2hL085ZhoUYOVw%2BHkT6%2Bs8G2nD50Xh28qE%2BM6QzmHg32deoV%0AwA%3D%3D%0A)

</Accordion>

<Accordion title="Step 2: Set it up">

In this guide, we will focus on creating a "Schedule task". After you click on the [Schedule Task] button, you need to fill in a few fields.

![Task settings](https://downloads.intercomcdn.eu/i/o/14257517/50fb3f2be560b0f25e40b3da/Capture-2024-05-30-161216.png?expires=1776771000&signature=a4b6dcc0598c78834ad8a199ecdcc214a4b578296fd8c0281ab59b0222a0ce4a&req=0dFtwF%2F%2FrTBk2hL085ZhofwMx0ssJgtXRASBZzNpo3R2D8ZtVi5ux7aehX07%0A%2Bg%3D%3D%0A)

**_2.1. Choose the owner of the task_**

The first thing you need to do after creating the task is make sure the owner is the same as the owner of the match. To do that, select [ID] in the [Match Owner] box and [Match: Owner (User)] in the [With] box after that. That way, the task will be assigned to this person.

![Matching the owner setting](https://downloads.intercomcdn.eu/i/o/14257568/233e4371dd887388c54d01ac/Capture-2024-05-30-161356.png?expires=1776771000&signature=305da9e5386f72d838b78dcfd812b2ee16cccd34269eb0e93d921018866508c0&req=0dFtwF%2F%2Fqj9k2hL085ZhoSL45xDt%2FSKHjXGnSBJ11eB9DBwl1pP%2BiHdnGt12%0ApQ%3D%3D%0A)

**_2.2. Set a start and a due date_**

Depending on when you want the task to start and be completed, you can set a [Start date] as well as a [Due Date].

![Start and due date options](https://downloads.intercomcdn.eu/i/o/14257760/c9b56b19ef85f124192cca07/Capture-2024-05-30-161603.png?expires=1776771000&signature=e7699689bb8d47a81965fcfb6a6fdd7dc7c9fd1e3f4596026c2c1969594946ba&req=0dFtwF%2F9qjdk2hL085ZhoYcwmw8G%2FC%2BuNHVXsuBhYb91bDaIZ5IGrYUJdzbg%0Ajw%3D%3D%0A)

**_2.3. Include a subject_**

In the [Subject] area, you can add variables. There are two ways to do that. The first one is to open the variables on the right via the **\{\}** button. Then type any keyword of the variable to find it quicker, and simply click on it to copy.

![Variables list](https://downloads.intercomcdn.eu/i/o/14258189/faf1770138d516c07e950699/Capture-2024-05-30-161900.png?expires=1776771000&signature=c93d4e85c9e321a2f54c673bffadb4002e497d24ff04cd832306721b7f6cd6e9&req=0dFtwFD7pD5k2hL085ZhoS1B1nCd5qA133opynxMhZX2olFyqjVYhGZgkzX%2B%0ARw%3D%3D%0A)

The other way to do this is to type "\{\{" followed by the variable's name in the subject field.

![Finding a variable in the subject field](https://downloads.intercomcdn.eu/i/o/14258960/55fd39ef267de38e8e9a07ae/Capture-2024-05-30-162817.png?expires=1776771000&signature=4ea4a4e4f31b846a58e09dc84060204c7be6151ca2fd663a31b480b91ca56c29&req=0dFtwFDzqjdk2hL085ZhobOH4Iz3uv%2FqLpEoVA2Ws%2BM1IUEZG3PcEkAGXQcZ%0A4w%3D%3D%0A)

For example, if you want to check if the candidate is happy, you need to type "Check if \{\{CandidateFirstName is happy".

![Subject example](https://downloads.intercomcdn.eu/i/o/14259452/4e36eb25a4139c0734ed7e6d/Capture-2024-05-30-164208.png?expires=1776771000&signature=f8fd3435a90d82261b614e6d94bbbbb89614e43ae9d1d04ec1e0c72b5c02704f&req=0dFtwFH%2BqTVk2hL085ZhoT2tiZpkKgrchV%2FK3OwtLqtAl6PawdS0a53R59Vw%0AWw%3D%3D%0A)

**_2.4. Write a description_**

Write a brief description of the task - this shouldn't be longer than one sentence.

![Description example](https://downloads.intercomcdn.eu/i/o/14259547/46e35f3789a820bcf3ea6015/Capture-2024-05-30-164426.png?expires=1776771000&signature=cbb108404276863096e852c53be25917b2d3e75bf8e0d352faef99fe6e816b87&req=0dFtwFH%2FqDBk2hL085Zhod%2FKRCmr%2Fba7lhNA6ZPpno%2FwgMDbcQPJMob7cVba%0A6A%3D%3D%0A)

**_2.5. Select the status of the task_**

It can be:

- To do
- In progress
- Done
- Screening accepted
- Screening declined

![Task status options](https://downloads.intercomcdn.eu/i/o/14259586/69ec8ee2d74a51b23c8d1113/Capture-2024-05-30-164542.png?expires=1776771000&signature=135b626a0d9e39b0c66068882877186908a86e311af297554f91f22e36c7d74c&req=0dFtwFH%2FpDFk2hL085ZhocnNgqzShTV1rVLey6%2BjhaWbBcE5BKiak4MUxmfy%0AAA%3D%3D%0A)

**_2.6. Add expiration date or a delay_**

This is optional, so if you don't want to add them, you can skip this part.

![Expiry date and delay settings](https://downloads.intercomcdn.eu/i/o/14259904/0fbafd1f2b0285a64479602a/image.png?expires=1776771000&signature=1fc6282e879f5891fd4227ed06e1fd7c918f8363bb769e15ebfe96a59f7c9260&req=0dFtwFHzrDNk2hL085ZhoatuJgxBKU7OhX4110UiEoRg8p6%2FgHYz4nWcCQIQ%0ALA%3D%3D%0A)

**_2.6.1. Setting an expiration date for the task_**

![Expiry date options](https://downloads.intercomcdn.eu/i/o/14259813/8e48bc16d3c48a958afc6634/image.png?expires=1776771000&signature=11ca3e5b1d9e08a4d622b0d40401cfe3051d22db8bd400a0dd7682a3e77ae48b&req=0dFtwFHyrTRk2hL085ZhoVeG3V7b5qvP4qW6TDqMq%2FIpc9pjQ7eSpLVkc%2B2n%0AQQ%3D%3D%0A)

**_2.6.2. Setting a delay for the task_**

![Delay options](https://downloads.intercomcdn.eu/i/o/14259791/8f21296680602c95f1f17881/image.png?expires=1776771000&signature=9832c8035f6d7b5260ac2236df9460a7c2d6072d4fc50b8ed2150e98a1a187e6&req=0dFtwFH9pTZk2hL085ZhoQvdT7DTIuZWaeWg6rx%2Fy8Ff7whyePPKbxjb2XYZ%0AhA%3D%3D%0A)

**_2.7. Match the candidate_**

Make sure to match the correct candidate to this task so it also matches the candidate's file in Carerix.

![Match Candidate field](https://downloads.intercomcdn.eu/i/o/14259936/480fe4c91cbb9559a11e05e4/image.png?expires=1776771000&signature=a7c6ce311e8234989b5800ed16a2b7a8ed3e795f0ba9cd78b0eec3d6b3f0afa5&req=0dFtwFHzrzFk2hL085ZhoWl5lhmO2H%2BojsR4iCmFOn2Yl%2FvlzzWs8FgYlZDK%0Acg%3D%3D%0A)

**_2.8. Save the task_**

Click on [Save Task] at the bottom of the Building Blocks menu.

![Save Task button](https://downloads.intercomcdn.eu/i/o/14259968/dbb80d93104a1b503f8aed0a/image.png?expires=1776771000&signature=b9f332300fa58dcacbdf51392c687e945b769bdb703a8e1aaeda23a07e2f7563&req=0dFtwFHzqj9k2hL085ZhoX1Y5nt%2BSKyzkkw8tOLIHruEoRWs2kqNH%2B9uoHMW%0AHg%3D%3D%0A)

</Accordion>

<Accordion title="Step 3: Review the task">

Review the summary of the task in the automation.

![Task summary](https://downloads.intercomcdn.eu/i/o/14260048/bcb1011ee91b619519ed024e/image.png?expires=1776771000&signature=8c3760cb05baee90442a58d0e577c9ee76515b0e3f2a0ae238c200672d50e5df&req=0dFtw1j6qD9k2hL085ZhoVNmr9SqcSW3RmY5zl4Gc6uq18qR%2FHxPd4rinPRc%0AaA%3D%3D%0A)
