_Welcome to our comprehensive guide on how to create a "Schedule task" in Carerix with HireData. Follow the instructions below to learn the process so you can manage it yourself. At the end, you will also find a video tutorial for additional support._

You should already have created an automation for Carerix - see how to do this [here](/apps/carerix/create-a-carerix-automation-in-hiredata).

## Step 1: Create a task

Click the [+] to open the Building Blocks menu, then on [Task] and finally select the [Carerix] app. You will see all of the automated tasks that we currently support through Carerix:

- Add Activity or Item
- Update Field(s)
- Schedule task

Select the task you want to create.

![Building Blocks menu with Carerix automated tasks](/images/intercom/244c3d20ce45.png)

## Step 2: Set it up

In this guide, we will focus on creating a "Schedule task". After you click on the [Schedule Task] button, you need to fill in a few fields.

![Task settings](/images/intercom/75885f020ae2.png)

**_2.1. Choose the owner of the task_**

The first thing you need to do after creating the task is make sure the owner is the same as the owner of the match. To do that, select [ID] in the [Match Owner] box and [Match: Owner (User)] in the [With] box after that. That way, the task will be assigned to this person.

![Matching the owner setting](/images/intercom/720df4ec3271.png)

**_2.2. Set a start and a due date_**

Depending on when you want the task to start and be completed, you can set a [Start date] as well as a [Due Date].

![Start and due date options](/images/intercom/898d7cee1844.png)

**_2.3. Include a subject_**

In the [Subject] area, you can add variables. There are two ways to do that. The first one is to open the variables on the right via the **\{\}** button. Then type any keyword of the variable to find it quicker, and simply click on it to copy.

![Variables list](/images/intercom/16948b32fa7d.png)

The other way to do this is to type "\{\{" followed by the variable's name in the subject field.

![Finding a variable in the subject field](/images/intercom/9520268e2ce8.png)

For example, if you want to check if the candidate is happy, you need to type "Check if \{\{CandidateFirstName is happy".

![Subject example](/images/intercom/35af7e599608.png)

**_2.4. Write a description_**

Write a brief description of the task - this shouldn't be longer than one sentence.

![Description example](/images/intercom/2849eab1e7a5.png)

**_2.5. Select the status of the task_**

It can be:

- To do
- In progress
- Done
- Screening accepted
- Screening declined

![Task status options](/images/intercom/d0e5894aa343.png)

**_2.6. Add expiration date or a delay_**

This is optional, so if you don't want to add them, you can skip this part.

![Expiry date and delay settings](/images/intercom/b17544b14cd9.png)

**_2.6.1. Setting an expiration date for the task_**

![Expiry date options](/images/intercom/17be4d5d7a8a.png)

**_2.6.2. Setting a delay for the task_**

![Delay options](/images/intercom/93b6beee5bea.png)

**_2.7. Match the candidate_**

Make sure to match the correct candidate to this task so it also matches the candidate's file in Carerix.

![Match Candidate field](/images/intercom/d9a4c4b18811.png)

**_2.8. Save the task_**

Click on [Save Task] at the bottom of the Building Blocks menu.

![Save Task button](/images/intercom/a468ea5bb9fc.png)

## Step 3: Review the task

Review the summary of the task in the automation.

![Task summary](/images/intercom/870bb021e218.png)
