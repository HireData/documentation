_Welcome to our comprehensive guide on how to log email messages from HireData to Carerix. This ensures that all email communication is recorded in Carerix, making it easier to track interactions with clients and candidates. Continue on to learn how to set up an automation in HireData that logs sent emails into Carerix._

##

<Accordion title="Step 1: Create a New Automation">

By integrating HireData with Carerix, you can automatically create email records in Carerix whenever an email is sent. The automation captures relevant details such as sender, recipient, subject, body content, and when the message has been sent, ensuring that every email is properly logged.

### 1.1. Navigate to the Automations section in HireData and click on [Create New Automation]

### 1.2. Select the "Log email in Carerix" template

![Log email in Carerix template](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42396003/96d7355641f3cf3a562c3fb17aa6/Screenshot+2025-03-21+at+17_20_20.png?expires=1776771000&signature=45345e4d13a7e6c432a0541811c8ddec5687224ed14a33d00de60b070d0fac2f&req=1NdszF76rDRk2hL085ZhobVRH6rDWAoVH5C13%2Bcd3V7K6jcp%2BqDVDz8ziXyZ%0A%2Bg%3D%3D%0A)

</Accordion>

<Accordion title="Step 2: Add a Trigger">

### 2.1. Review the app connection

It will be automatically selected, but we recommend making sure it's the correct one in case you have multiple connections with Carerix.

### 2.2. Select Object

The [Message] object will also be automatically configured, but this can be edited when you click on the [Edit] button. Here are the other object options we support in this case:

![Objects](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42401482/b9d078f945e564ae3bc3ebe6aaaa/Screenshot+2025-03-21+at+17_40_18.png?expires=1776771000&signature=b4bda6bc3dfa43a824e1fa3ef809bbeb7f5656333fffd6bca6e856088c6eeb51&req=1NdrxVn%2BpDVk2hL085Zhocl%2Bn5H0uPwjQK14KYBy4cOKuwzji3BlMdSAunsZ%0Avw%3D%3D%0A)

### 2.3. Set a trigger

Set the trigger to [Message: Sent] to capture emails as soon as they are sent.

![Trigger set up](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42401972/2f89c9fd9235d328991fdf547c9c/Screenshot+2025-03-21+at+17_20_57.png?expires=1776771000&signature=bde45b8224170e4fc113806290c5d480ba19f50992c2b68ed911b2823e84181b&req=1NdrxVnzqzVk2hL085ZhoZacO6nAa5Aon3%2F0HBtM5%2FW%2FGxC6Zl5wr7W0naeW%0AoQ%3D%3D%0A)

### 2.4. Add filters (optional)

Adding a filter will result in better segmentation depending on your needs. For this example, we want to filter out all messages that are sent via other channels, so only message that are sent via email will be added to Carerix.

![Setting up a filter](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42413297/4a26e6b85bf2adf54439bd1a90fe/Screenshot+2025-03-21+at+17_21_48.png?expires=1776771000&signature=6cc50aa8756e8385a60e9cbc14066ebaa395d59304f8244ab18153326e544057&req=1NdrxFv4pTBk2hL085ZhocSYTYJxQNRes2lx5rOEj6VNoxro6xE0GX5bLaKg%0Apg%3D%3D%0A)

</Accordion>

<Accordion title="Step 3: Set up the [Add Email to Carerix] Task">

This task is a part of the automation template where you can fill in the required fields to ensure the email data is logged correctly:

### 3.1. Select the Object you would like to create in Carerix, e.g. email, note, etc.

You can select any of these objects to be added to your app:

- **Appointment:** To schedule meetings or interviews
- **Candidate:** To add a new candidate to the system
- **Email:** To log email communications
- **Match:** To connect candidates with job opportunities
- **Note:** To record conversation details or observations
- **Task:** For follow-up actions or to-do items

![Object selection](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42416121/32e4484b24506a044297005140fd/Screenshot+2025-03-21+at+17_57_03.png?expires=1776771000&signature=111924c591305f774c8a8ef1ba23185dcee83410e10d40105873ecb82bf59bc6&req=1NdrxF77rjZk2hL085ZhodcV4%2Bo68dA8%2BLUaKo5A1%2FNEQzyLBkcOqPW3up4w%0AnA%3D%3D%0A)

### 3.2. Select the sender and the recipient email addresses to log in Carerix

![Sender and receiver fields](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42416518/4631b2583c3b33ba25e45ab75c8f/Screenshot+2025-03-21+at+17_58_31.png?expires=1776771000&signature=9a8bd8c2f406b4079e38f1e5a8004f11c1135090f83e9145f5937f2794b2f800&req=1NdrxF7%2FrT9k2hL085ZhoaXMIMGxvn8fZNGROfNGDaEYjvkSa2eq380%2Bh4Tp%0Amg%3D%3D%0A)

### 3.3. Then ensure the correct email content is recorded

- Subject
- Body

![Email content fields](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42422610/3c360b03967824b8e3ebbe10d6ce/Screenshot+2025-03-21+at+18_01_36.png?expires=1776771000&signature=2257412ef6673eb6e2689308bb62fd093672fdd509175520ca02fab2613bd005&req=1Ndrx1r8rTdk2hL085ZhoW6c0DCA52xrjWZq3Smk5zYDF%2Fgys4yOy0mZC5o8%0AGA%3D%3D%0A)

### 3.4. Set the status to any of the following options depending on the emails you would like to log

![Setting up an email status](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42501386/2bd20e8c5d65c17a63d143d1c3e7/Screenshot+2025-03-21+at+18_02_33.png?expires=1776771000&signature=989cc0dd71fb722f17f365bda4563e9bd2fa42006136348b510e774233e25c81&req=1NdqxVn5pDFk2hL085ZhoYoO67eFM6PqpZTlFzqnHkZBwE%2BmK89uRvTs5Eah%0AAQ%3D%3D%0A)

### 3.5. Add a Send On condition

For example, use [Message: Sent At] to log the timestamp of the email:

![Send On field](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/42502640/8f63f7ad851b2c1fdf9784124500/Screenshot+2025-03-21+at+23_10_29.png?expires=1776771000&signature=53753d0397e8e0adad20ea7d47f5c12f54e200411d627071f2c034bc48da6c9a&req=1NdqxVr8qDdk2hL085Zhoc9l%2BkJdpB%2FMCNlkohvPKtl66%2FdKtEZ9HOaTQRuB%0AgA%3D%3D%0A)
