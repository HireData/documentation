_Welcome to our detailed guide on connecting your Recruitee account with HireData. This tutorial is designed to be user-friendly and efficient, helping you to leverage the full potential of Recruitee within your HireData automations. For a visual guide, be sure to watch our step-by-step **Connect with Recruitee Video** at the end._

## Step 1: Navigating to the Recruitee App

Begin by navigating to the [Apps section](https://app.hiredata.com/settings/apps) within HireData. You can find this under Settings &gt; Apps. Here, you’ll see the option to [integrate with Recruitee.](https://app.hiredata.com/apps/recruitee) This initial step is simple but essential for the integration process.

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/65643328/ed009f979b4c2d6d334861abbc59/Schermafbeelding%2B2025-09-19%2B100347.png?expires=1776771000&signature=ae01837006d10df5a5372780098a7a3da6a6ccefa4882887fdf5aa7627aaccec&req=1tBpwVv5rj9k2hL085ZhoXDA9BenNlzwYexLd3IXSM%2FHb%2B4HvyWizYi1zGDc%0A%2BA%3D%3D%0A)

## Step 2: Creating a Connecting with Recruitee

Open the app and click **Connect to Recruitee** to start the integration process. This will allow HireData to connect with your Recruitee account.

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/65646660/e9eaf1638899de95c6e8981ea183/Schermafbeelding%2B2025-09-19%2B102104.png?expires=1776771000&signature=f49c6724b7a5ea9c43a46523fe5ea431fe079270661c798a035cb387f6f8c325&req=1tBpwV78qjdk2hL085ZhoV%2BxQFi%2Fc8oSVHKaKWk97xXuQGn4nRFGn9Mbi1hM%0A3g%3D%3D%0A)

### 2.1. Create API token

Click on '**here'** to create a new API token in Recruitee.

Make sure your user has these roles:

- Basic permissions → Collaborate on candidates, Join hiring teams
- Jobs → View all existing jobs data
- Hires → View hired candidates
- Add-ons → Manage API tokens

Now copy the generated API token and paste it below to connect Recruitee with HireData.

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/65646659/c71a6510734b8a14f61bfa667994/image.png?expires=1776771000&signature=837bfec9c95a29feaf0bdf1f451364093ad8a859a66f3941c4a83f35ddfa8eb2&req=1tBpwV78qT5k2hL085ZhoWwqYtpYVxsLMNYGg%2B89DTl8kyxAx47br%2BAPY%2BGj%0AUQ%3D%3D%0A)

### 2.2. Copy Your Recruitee Company ID

Under **Current company details in Recruitee**, you’ll see your **Company ID**. Simply copy this ID and paste it into the HireData field.

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/65646661/ab76f65d1568eeb1f6e661b2129c/image.png?expires=1776771000&signature=d9a46948fd3104234278ded3edfe3635762be7aa6aa2409fdfffcdc14807b5da&req=1tBpwV78qjZk2hL085ZhocLpH5DRh5pG%2BVOZku3612F8othXgTR5Ie5rhRLg%0A7w%3D%3D%0A)

Once filled in, click [Continue] to complete the connection with Recruitee.

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/66842172/38b2d50741859e12b60b7d682ed1/image.png?expires=1776771000&signature=e605e0319cc9e6f2406059662dddac0bd64f54f15eb0856f9f1a84cc9c66afac&req=1tNnwVr7qzVk2hL085ZhocMJ62qh4IGtuW2PI7Qb%2FiPOs2aVxtLk6jjFhoim%0Agw%3D%3D%0A)

## Step 3: Recruitee Connected!

After you've clicked [Continue], you've finished setting up the connection between Recruitee and HireData. When it's the first time you've set up this connection, HireData might need a few minutes to sync data from your Recruitee database. After that, you're all set to [set up your first Recruitee automation](https://help.hiredata.com/en/articles/412041-create-a-recruitee-automation-in-hiredata) in HireData.

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/66841878/6966e008c570887bd3739a2c1e39/image.png?expires=1776771000&signature=831996838c92de75a61e0b8952fea08edda5c7c31a32b6e263c17fb52b7b6699&req=1tNnwVnyqz9k2hL085ZhoewMjp0q3WrE0z0334bGgXiO%2FwiqtTjAEp0MpLsZ%0AxQ%3D%3D%0A)
