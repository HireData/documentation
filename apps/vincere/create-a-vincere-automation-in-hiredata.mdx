_Welcome to our tutorial on setting up Vincere automation within HireData. With this integration, you can automate tasks such as sending Candidate NPS surveys, thereby improving candidate experiences and streamlining your recruitment processes. Follow along as we demonstrate each step to set up this automation efficiently._

## Step 1: Setting up the Trigger

After you've [set up the connection with Vincere](https://help.hiredata.com/en/articles/126717-set-up-a-connection-with-vincere), the first step in setting up Vincere automation is to define the trigger. A trigger makes sure HireData "listens" to certain events occurring within Vincere. Don't forget to select all relationships you'll need later on, e.g. select a candidate when you want to send a survey to them in a further step within the automation. Here's how to do it:

_Example Trigger: Application Status Changed_

_Select the related objects you'll need later on in the automation, e.g. a user you want to be the sender of a message and the recipient(s) of that message_
