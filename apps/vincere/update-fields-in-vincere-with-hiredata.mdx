_Welcome to our step-by-step guide on how to update one or more fields in different objects in Vincere with HireData. Follow this simple but helpful guide to understand how to update any field yourself._

Before updating the fields in the object, you should have already created an [automation in Vincere](/apps/vincere/create-a-vincere-automation-in-hiredata).

## Step 1: Open the building blocks menu

First, log in to your HireData account and open the automation you want to edit. Then click on the plus icon [+] at the end of the automation to open the Building Blocks menu on the right side of the screen. After you have opened the menu, select [Update].

![Building Blocks menu](/images/intercom/51931242-Screenshot-2025-06-02-at-09_16_11.png)

## Step 2: Select the Object you want to edit

Choose the object you want to edit, which currently can be one of these:

- Candidate
- Company
- Contact
- Job
- Meeting
- Task

_For this example, we have selected the candidate:_

![Selecting an object to be updated](/images/intercom/51931309-Screenshot-2025-06-02-at-09_19_18.png)

## Step 3: Select an Identifier

Now you have to select an identifier, which will be used to find the object that you want to update.

![Selecting an object identifier](/images/intercom/51931462-Screenshot-2025-06-02-at-09_19_42.png)

For this guide, we will use the [ID] as an identifier and the [Candidate: ID] as the value we want to track. In other words, we will use the Candidate ID to track the candidate we want to update in the automation.

**_The displayed values will be different depending on the selected object and identifier._**

![Selecting object identifier](/images/intercom/51931552-Screenshot-2025-06-02-at-09_19_50.png)

## Step 4: Update the fields

Now, you can select all the fields you want to update—you can select as many as you want, just use the search bar to find them.

![Selecting field to be updated](/images/intercom/51931636-Screenshot-2025-06-02-at-09_20_14.png)

Edit the fields and click the [Save Task] button at the bottom of the page.

![Saving the update fields task](/images/intercom/51931666-Screenshot-2025-06-02-at-09_22_04.png)

## Step 5: Review

Last but not least, review the task pop-up, which will appear at the end of the automation, to ensure everything runs smoothly.

![Update field task](/images/intercom/51931722-Screenshot-2025-06-02-at-09_22_36.png)
