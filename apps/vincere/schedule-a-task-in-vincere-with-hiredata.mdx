_The Vincere Scheduled Task action allows you to automatically create scheduled tasks in Vincere whenever a specific trigger event occurs in a HireData automation workflow. This is useful for automating reminders and follow-up activities, ensuring no task is overlooked. Follow along as we demonstrate how to use this task._

## Step 1: Create or edit an automation

In your HireData account, navigate to the [Automations] section and either create a new workflow or open an existing one by clicking on [+New Automation].

## Step 3: Fill in he task details

- **Type:** Choose the Item Type you would like the Task to correspond to. This can be:
  - Candidate - A task relating to the Job Seeker
  - Contact - A task tied to the Client Contact
  - Job - A task relating to a Job Order or Vacancy
  - Personal - A task for your own use, such as reminders or admin duties
  - Project - A task associated with a larger project or campaign

![](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/72656330/fb15090057f6d33eac36feb2ffa8/image.png?expires=1776771000&signature=629e4a3fbdbbee9c9ebfe56f597ebc16a611489c1dd78089138b380ecaac0417&req=19dpwF75rzdk2hL085ZhoaPU3rux%2FT2kXiyEf4Cf7kQCMpOcUyL%2FkFiQnBfD%0ATQ%3D%3D%0A)

- **Subject:** Enter a clear and descriptive subject for the task (e.g., "Test project, local")
- **Description:** Provide a detailed description of the task
- **Start/End Date:** Set the task's start and end dates. You can also set a delay if you wish
- **Reminders:** Add a reminder to ensure the task is not forgotten
- **Assignee:** Assign the task to the relevant user in Vincere
![Task configuration](https://downloads.intercomcdn.eu/i/o/lk3ftrxl/70620488/51505b6592acb357cdad63840de6/image.png?expires=1776771000&signature=0664a34107423c41bb96eda20e5ee3d09f3471c3cd4dba9a008cccbfac5e3cf0&req=19Vpx1j%2BpD9k2hL085ZhoUgXgG5Pi%2FpJx0M3QvtAbXgV%2Fk31n50GQvVUeP%2B9%0Acw%3D%3D%0A)

## Step 4: Save and test

Click [Save Task] to finalise the task configuration and [run the automation workflow to test it](https://help.hiredata.com/en/articles/311523-how-to-test-an-automation).
