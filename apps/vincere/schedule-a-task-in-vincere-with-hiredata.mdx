title: "Schedule a task in Vincere with HireData"
description: "Learn how to schedule a task within Vincere using HireData"
---

_The Vincere Scheduled Task action allows you to automatically create scheduled tasks in Vincere whenever a specific trigger event occurs in a HireData automation workflow. This is useful for automating reminders and follow-up activities, ensuring no task is overlooked. Follow along as we demonstrate how to use this task._

## Step 1: Create or edit an automation

In your HireData account, navigate to the [Automations] section and either create a new workflow or open an existing one by clicking on [+New Automation].

![Adding new automation in HireData](/images/intercom/70618331-image.png)

## Step 2: Add the action

### 2.1. Click on [Blank Automation] to start a new automation from scratch

Selecting Blank Automation will create a new automation

![Start a blank automation](/images/intercom/73792029-image.png)

### 2.2. Click in the First Building Block/Trigger to bring up the App Panel and select [Vincere]

Clicking on the First Building block you'll be able to choose which App to link this to

![First step to creating a Trigger by selecting the brand](/images/intercom/72641866-image.png)

### 2.3. Select the Trigger for the Automation

Follow the setup steps to choose how you would like this automation to trigger and click [Save].

Selecting the Object value for the app

![Trigger Setup Choosing Object](/images/intercom/72645601-image.png)

Choosing the timing in which you would like the trigger to adhere to

![Trigger Setup Timing](/images/intercom/72645827-image.png)

Checking that everything is correct and saving the trigger

![Overview of Trigger](/images/intercom/72646002-image.png)

### 2.4. Click on the 2nd Building Block to open the action panel and click on [Add] to create a new Task

![](/images/intercom/72648002-image.png)

### 2.5. Navigate until you see the Vincere logo along with the text [Task]

![Add Task](/images/intercom/72648586-image.png)

## Step 3: Fill in he task details

- **Type:** Choose the Item Type you would like the Task to correspond to. This can be:
  - Candidate - A task relating to the Job Seeker
  - Contact - A task tied to the Client Contact
  - Job - A task relating to a Job Order or Vacancy
  - Personal - A task for your own use, such as reminders or admin duties
  - Project - A task associated with a larger project or campaign

![](/images/intercom/72656330-image.png)

- **Subject:** Enter a clear and descriptive subject for the task (e.g., "Test project, local")
- **Description:** Provide a detailed description of the task
- **Start/End Date:** Set the task's start and end dates. You can also set a delay if you wish
- **Reminders:** Add a reminder to ensure the task is not forgotten
- **Assignee:** Assign the task to the relevant user in Vincere
![Task configuration](/images/intercom/70620488-image.png)

## Step 4: Save and test

Click [Save Task] to finalise the task configuration and [run the automation workflow to test it](/automations/how-to-test-an-automation).

  - Navigate to Task History to verify it was created![Track task in HireData](/images/intercom/70620685-image.png)

- Vincere
  - Email View &gt; Task Tab
![Track Task in Vincere](/images/intercom/70620741-image.png)
